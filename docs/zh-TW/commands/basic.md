# 基礎功能

## 說明

查看所有可用指令的清單（圖片形式）。

**指令：** `/帮助`

---

## 今日運勢

查看今日 maimai 運勢，每天隨機生成。

**指令：** `/今日舞萌`

---

## 隨機歌曲

隨機推薦一首曲目，可按類型、難度顏色、等級篩選。

**指令：** `/隨機歌曲 [類型] [顏色] [等級]`

**參數：**

| 參數 | 說明 | 可選值 |
|------|------|--------|
| 類型 | 曲目類型（可選） | `dx`、`sd` |
| 顏色 | 難度顏色（可選） | `綠`、`黃`、`紅`、`紫`、`白` |
| 等級 | 等級（可選） | `12`、`13+` 等 |

**示例：**

| 指令 |
|------|
| `/隨機歌曲 dx 紫 14` |
| `/隨機歌曲 13+` |
| `/隨機歌曲 sd` |

::: tip
參數順序不固定，輸入時可任意組合，例如 `/隨機歌曲 14 dx` 和 `/隨機歌曲 dx 14` 效果相同。
:::

## 切換語言

切換 Bot 的回覆語言。

**指令：** `/lang [語言代碼]`

**參數：**

| 語言代碼 | 說明 |
|----------|------|
| `zh`（或 `cn`） | 簡體中文（預設） |
| `zh-Hant`（或 `tw`） | 繁體中文（台灣） |
| `zh-Hant-HK`（或 `hk`） | 繁體中文（香港） |
| `en`（或 `english`） | English |
| `ja`（或 `jp`） | 日本語 |

不帶參數執行時，顯示目前語言設定。

**示例：**

| 指令 | 說明 |
|------|------|
| `/lang` | 查看目前語言 |
| `/lang zh-Hant` | 切換到繁體中文 |
| `/lang en` | 切換到英文 |
