# 基礎功能

## 說明

查看所有可用指令的清單。

**指令：** `/帮助`

---

## 今日運勢

查看今日 maimai 運勢，每天隨機生成。

**指令：** `/今日舞萌`

---

## 隨機歌曲

隨機推薦一首曲目，可按類型、難度顏色、等級篩選。

**指令：** `/随机歌曲 [類型] [顏色] [等級]`

**參數：**

| 參數 | 說明 | 可選值 |
|------|------|--------|
| 類型 | 曲目類型（可選） | `dx`、`sd`（或 `标准`） |
| 顏色 | 難度顏色（可選） | `绿`（綠）、`黄`（黃）、`红`（紅）、`紫`、`白` |
| 等級 | 等級（可選） | `12`、`13+` 等 |

**示例：**

| 指令 |
|------|
| `/随机歌曲 dx 紫 14` |
| `/随机歌曲 13+` |
| `/随机歌曲 sd` |

::: tip
參數順序不固定，輸入時可任意組合，例如 `/随机歌曲 14 dx` 和 `/随机歌曲 dx 14` 效果相同。
:::
