# 進度查詢

## 等級進度 / 牌子進度

查詢指定等級或版本牌子下的達成進度，支援按完成狀態篩選和分頁。

**指令：** `/進度 <等級/定數/牌子> <評價/評級> [已完成/未完成/未開始] [頁數]`

### 等級進度模式

**參數：**

| 參數 | 說明 | 示例 |
|------|------|------|
| 等級或定數 | `12`、`13+`、`14.0` 等 | `13+` |
| 評價 | 目標達成率、FC/AP 或評價名 | `99.5`、`fc`、`ap`、`SSS` |
| 已完成/未完成/未開始 | 可選，篩選完成狀態 | `已完成`、`未完成`、`未開始` |
| 頁數 | 可選，預設第 1 頁 | `2` |

**分數評價對照：**

| 評價 | 達成率 |
|------|--------|
| `S` | ≥ 97.0000% |
| `S+` | ≥ 98.0000% |
| `SS` | ≥ 99.0000% |
| `SS+` | ≥ 99.5000% |
| `SSS` | ≥ 100.0000% |
| `SSS+` | = 101.0000% |

**同步/FC 評價：** `fc`、`fc+`、`ap`、`ap+`、`fs`、`fs+`、`fdx`、`fdx+`

**示例：**

| 指令 |
|------|
| `/進度 13+ SSS` |
| `/進度 14 fc` |
| `/進度 13+ SS 已完成` |
| `/進度 14 ap 未完成 2` |

### 版本牌子進度模式

查詢指定版本牌子的通關進度（如「丸將」「真極」等）。

**格式：** `/進度 <牌子字> <評級>`

| 評級 | 說明 |
|------|------|
| `极` | FC 所有曲目 |
| `将` | SSS 所有曲目 |
| `神` | AP 所有曲目 |
| `舞舞` | FDX 所有曲目 |

**示例：**

| 指令 |
|------|
| `/進度 丸 将` |
| `/進度 真 极` |
| `/進度 鏡 神 2` |

::: tip
牌子字和評級可以連寫，例如 `/進度 丸将` 與 `/進度 丸 将` 效果相同。
:::

## 分類進度

按歌曲分類（動漫、東方 Project 等）查詢進度。

**指令：** `/分類進度 <分類> <評價/評級> [已完成/未完成/未開始] [頁數]`

**參數：**

| 參數 | 說明 | 示例 |
|------|------|------|
| 分類 | 歌曲類別名稱 | `動漫`、`東方`、`舞萌`、`V家`、`遊戲`、`音擊`、`宴` |
| 評價/評級 | 目標達成率或評級 | `99.5`、`fc`、`ap`、`SSS` |
| 已完成/未完成/未開始 | 可選，篩選完成狀態 | `已完成`、`未完成`、`未開始` |
| 頁數 | 可選，預設第 1 頁 | `2` |

**示例：**

| 指令 |
|------|
| `/分類進度 動漫 SSS` |
| `/分類進度 東方 SS 2` |
| `/分類進度 舞萌 fc 已完成 2` |

## 分數列表

查詢你在指定等級/定數下所有曲目的分數列表，支援多維篩選與排序。

**指令：** `/分數列表 [等級或定數] [篩選條件...] [排序] [頁數]`

**篩選條件（可疊加）：**

| 類型 | 可用值 |
|------|--------|
| Combo | `ap`、`fc`、`fc+`/`fcp`、`寸`（寸止） |
| Sync | `fs`、`fs+`/`fsp`、`fdx`、`fdx+`/`fdxp` |
| 評價 | `sss+`、`sss`、`ss+`、`ss`、`s+`、`s`、`a`、`aa`、`aaa`、`b`、`bb`、`bbb` 等 |
| 譜面類型 | `dx`、`sd` |
| 新舊版本 | `new`（新版）、`old`（舊版） |
| DX 星級 | `1star`～`6star` |

在篩選條件前加 `=` 表示精確匹配（僅適用於 Combo/Sync），如 `=fc` 僅顯示恰好 FC、未達到 FC+ 的曲目。

**排序：** `按X`（升序）或 `按X+`/`按X-`（指定方向）

可排序欄位：`ra`（Rating 貢獻）、`達成度`/`acc`、`dx`（DX 分）、`定數`/`ds`

**示例：**

| 指令 | 說明 |
|------|------|
| `/分數列表 13+` | 查詢所有 13+ 曲目分數 |
| `/分數列表 14 fc` | 篩選 FC（含 FC+）及以上 |
| `/分數列表 14 =fc` | 僅顯示恰好 FC（未達 FC+）的曲目 |
| `/分數列表 14 ap 2` | 篩選 AP 及以上，第 2 頁 |
| `/分數列表 14 dx sss` | 同時篩選 DX 譜和 SSS 及以上 |
| `/分數列表 14 按ra-` | 按 Rating 貢獻降序排列 |
| `/分數列表 14 1star` | 篩選 DX 星級為 1 星的曲目 |
