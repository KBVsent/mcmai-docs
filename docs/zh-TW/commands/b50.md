# Best 50 功能

## Best 50

查詢你的 Best 50 成績（Top 35 SD + Top 15 DX）。

**指令：** `/b50`

支援透過 `#使用者名稱` 查詢他人資料（需對方開啟查詢權限）。

**示例：**

| 指令 |
|------|
| `/b50` |
| `/b50 #某人` |

## PC50

查詢你的遊玩次數前 50 成績。

::: info 僅國服
此功能僅國服（CN）用戶可用。
:::

**指令：** `/pc50`

支援透過 `#使用者名稱` 查詢他人資料（需對方開啟查詢權限）。

---

## PK B50

以對方的 B50 曲目為基準，對比雙方在相同譜面上的成績差異。

**指令：** `/pk50 [#使用者名稱]`

- 指定對手：`/pk50 #Sakura`
- 不指定時：在目前群組內隨機匹配同服玩家

**示例：**

| 指令 | 說明 |
|------|------|
| `/pk50 #Sakura` | 與 Sakura PK |
| `/pk50` | 群內隨機匹配對手 |

---

## 更新 B50 資料

從 maimai NET 同步你的最新 B50 資料到本機快取。

**指令：** `/更新記錄`

::: tip 國服用戶透過掃碼更新
國服用戶使用舞萌 ｜ 中二 公眾號內「玩家二維碼」功能獲取 QR Code，長按 QR Code 解析後將含有 `SGWCMAID` 字樣的內容發送到 Bot，會自動觸發國服 B50 資料更新。
:::
