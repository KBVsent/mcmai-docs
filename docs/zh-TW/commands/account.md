# 帳號與伺服器

## 綁定 SEGA ID

將你的 QQ 帳號與 SEGA ID 關聯，用於查詢 maimai NET 資料。

只有國際服和日服用戶需要綁定帳號，國服用戶透過 MAID 直接更新資料。

**指令：** `/绑定`

**說明：** 執行後會生成一個專屬綁定連結（或 QR Code），在瀏覽器中開啟並完成授權即可。

::: warning 注意
每次執行 `/绑定` 會生成新的驗證碼，請及時完成綁定。
:::

## 切換伺服器

切換你目前綁定的遊戲區服。

**指令：** `/切服 <伺服器>`

**參數：**

| 參數 | 說明 |
|------|------|
| `cn` | 國服 |
| `jp` | 日服 |
| `intl` | 國際服 |

**示例：**

| 指令 |
|------|
| `/切服 jp` |
| `/切服 cn` |
| `/切服 intl` |

## 設定使用者名稱

設定一個便於他人查詢你資料的使用者名稱。

**指令：** `/用户名 [使用者名稱|QQ號|清除]`

**示例：**

| 指令 | 說明 |
|------|------|
| `/用户名 Sakura` | 設定使用者名稱為 Sakura |
| `/用户名 123456789` | 關聯 QQ 號為使用者名稱 |
| `/用户名` | 查看目前使用者名稱 |
| `/用户名 清除` | 清除使用者名稱 |

::: warning 注意
使用使用者名稱作為查詢參數時，使用者名稱前一定要加 `#`。

例如 `/b50 #Sakura`，而不是 `/b50 Sakura`。
:::

## 隱私設定

控制他人能否查詢你的遊戲資料。

**指令：** `/隐私设置 <動作> [目標]`

**動作：**

| 動作 | 說明 |
|------|------|
| `查看` | 查看目前隱私設定 |
| `开启` | 允許他人查詢指定目標 |
| `关闭` | 禁止他人查詢指定目標 |
| `重置` | 恢復指定目標為預設（全部允許） |

**目標：**

| 目標 | 說明 |
|------|------|
| `查询` | 總開關，控制所有查詢權限 |
| `b50查询` | Best 50 等各類 50 查詢 |
| `分数查询` | 成績（info）查詢 |
| `进度查询` | 完成表 / 進度表查詢 |

::: tip
關閉 `查询`（總開關）後，他人將無法查詢你的任何資料，優先權高於其他單項設定。
:::

**示例：**

| 指令 | 說明 |
|------|------|
| `/隐私设置 查看` | 查看目前各項隱私狀態 |
| `/隐私设置 关闭 查询` | 禁止他人查詢你的所有資料 |
| `/隐私设置 关闭 分数查询` | 單獨禁止他人查詢你的成績 |
| `/隐私设置 开启 b50查询` | 重新允許他人查詢你的 Best 50 |
| `/隐私设置 重置 查询` | 恢復所有查詢權限為預設值 |

## 好友管理

管理你的 McMai 好友清單，好友清單用於 PK 功能。

**指令：** `/好友 [動作] [參數]`

**別名：** `friend`、`好友管理`

**動作：**

| 動作 | 說明 | 參數 |
|------|------|------|
| `列表` | 查看好友清單 | — |
| `添加` | 新增好友 | `#使用者名稱` |
| `删除` | 刪除好友 | `#使用者名稱` 或槽位號 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/好友 列表` | 查看好友清單 |
| `/好友 添加 #Sakura` | 新增名為 Sakura 的好友 |
| `/好友 添加 #Sakura 1` | 在槽位號為 1 的位置覆蓋新增名為 Sakura 的好友 |
| `/好友 删除 1` | 刪除槽位號為 1 的好友 |
