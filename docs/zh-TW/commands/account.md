# 帳號與伺服器

## 綁定 SEGA ID

將你的 QQ 帳號與 SEGA ID 關聯，用於查詢 maimai NET 資料。

只有國際服和日服用戶需要綁定帳號，國服用戶透過 MAID 直接更新資料。

**指令：** `/綁定`

**說明：** 執行後會生成一個專屬綁定連結（或 QR Code），在瀏覽器中開啟並完成授權即可。

::: warning 注意
每次執行 `/綁定` 會生成新的驗證碼，請及時完成綁定。
:::

## 切換伺服器

切換你目前綁定的遊戲區服。

**指令：** `/切換伺服器 <伺服器>`

**參數：**

| 參數 | 說明 |
|------|------|
| `cn` | 國服 |
| `jp` | 日服 |
| `intl` | 國際服 |

**示例：**

| 指令 |
|------|
| `/切換伺服器 jp` |
| `/切換伺服器 cn` |
| `/切換伺服器 intl` |

## 設定使用者名稱

設定一個便於他人透過 `#使用者名稱` 查詢你資料的標識符。

**指令：** `/使用者名稱 [名稱|QQ號|清除|clearqq]`

**示例：**

| 指令 | 說明 |
|------|------|
| `/使用者名稱 Sakura` | 設定使用者名稱為 Sakura |
| `/使用者名稱` | 查看目前使用者名稱、查分器 ID 與綁定 QQ |
| `/使用者名稱 清除` | 清除使用者名稱 |
| `/使用者名稱 123456789` | 綁定 QQ 號（5～11 位數字） |
| `/使用者名稱 clearqq` | 解綁已綁定的 QQ 號 |

**查詢標識符說明：**

每位用戶都有一個系統自動分配的**查分器 ID**（`#mc` 開頭的短 ID），即使未設定使用者名稱也可使用它被他人查詢。設定使用者名稱後，他人可以透過 `#使用者名稱` 查詢你，也可以繼續使用查分器 ID。

::: warning 注意
在使用使用者名稱作為查詢參數時，使用者名稱前一定要加 `#`。

例如 `/b50 #Sakura`，而不是 `/b50 Sakura`。
:::

::: tip 審核機制
使用者名稱設定後需經過審核：

- **審核中**：可以用 `#使用者名稱` 被他人查詢，但顯示時使用者名稱會被部分遮蓋
- **審核通過**：使用者名稱完整顯示
- **審核不通過**：使用者名稱將被自動刪除，需重新設定

使用者名稱每 **3 天** 只能修改一次，請謹慎設定。
:::

## 隱私設定

控制他人能否查詢你的遊戲資料。

**指令：** `/隱私設定 <動作> [目標]`

**動作：**

| 動作 | 說明 |
|------|------|
| `查看` | 查看目前隱私設定 |
| `開啟` | 允許他人查詢指定目標 |
| `關閉` | 禁止他人查詢指定目標 |
| `重置` | 恢復指定目標為預設（全部允許） |

**目標：**

| 目標 | 說明 |
|------|------|
| `查詢` | 總開關，控制所有查詢權限 |
| `b50查詢` | Best 50 等各類 50 查詢 |
| `分數查詢` | 成績（info）查詢 |
| `進度查詢` | 完成表 / 進度表查詢 |

::: tip
關閉 `查詢`（總開關）後，他人將無法查詢你的任何資料，優先權高於其他單項設定。
:::

**示例：**

| 指令 | 說明 |
|------|------|
| `/隱私設定 查看` | 查看目前各項隱私狀態 |
| `/隱私設定 關閉 查詢` | 禁止他人查詢你的所有資料 |
| `/隱私設定 關閉 分數查詢` | 單獨禁止他人查詢你的成績 |
| `/隱私設定 開啟 b50查詢` | 重新允許他人查詢你的 Best 50 |
| `/隱私設定 重置 查詢` | 恢復所有查詢權限為預設值 |

## 好友管理

管理你的 McMai 好友清單，好友清單用於 PK 功能。

**指令：** `/好友 [動作] [參數]`

**別名：** `friend`、`好友管理`

**動作：**

| 動作 | 說明 | 參數 |
|------|------|------|
| `列表` | 查看好友清單 | — |
| `新增` | 新增好友 | `#使用者名稱` |
| `刪除` | 刪除好友 | `#使用者名稱` 或槽位號 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/好友 列表` | 查看好友清單 |
| `/好友 新增 #Sakura` | 新增名為 Sakura 的好友 |
| `/好友 新增 #Sakura 1` | 在槽位號為 1 的位置覆蓋新增名為 Sakura 的好友 |
| `/好友 刪除 1` | 刪除槽位號為 1 的好友 |
