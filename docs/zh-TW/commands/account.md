# 帳號與伺服器

## 綁定 SEGA ID

將你的 QQ 帳號與 SEGA ID 關聯，用於查詢 maimai NET 資料。

只有國際服和日服用戶需要綁定帳號，國服用戶透過 MAID 直接更新資料。

**指令：** `/绑定`

**說明：** 執行後會生成一個專屬綁定連結（或 QR Code），在瀏覽器中開啟並完成授權即可。

::: warning 注意
每次執行 `/绑定` 會生成新的驗證碼，請及時完成綁定。
:::

---

## 切換伺服器

切換你目前綁定的遊戲區服。

**指令：** `/切服 <伺服器>`

**參數：**

| 參數 | 說明 |
|------|------|
| `cn` | 國服 |
| `jp` | 日服 |
| `intl` | 國際服 |

**示例：**

| 指令 |
|------|
| `/切服 jp` |
| `/切服 cn` |
| `/切服 intl` |

## 設定使用者名稱

設定一個便於他人查詢你資料的使用者名稱。

**指令：** `/用户名 [使用者名稱|QQ號|清除]`

**示例：**

| 指令 | 說明 |
|------|------|
| `/用户名 Sakura` | 設定使用者名稱為 Sakura |
| `/用户名 123456789` | 關聯 QQ 號為使用者名稱 |
| `/用户名` | 查看目前使用者名稱 |
| `/用户名 清除` | 清除使用者名稱 |

::: warning 注意
使用使用者名稱作為查詢參數時，使用者名稱前一定要加 `#`。

例如 `/b50 #Sakura`，而不是 `/b50 Sakura`。
:::

## 隱私設定

控制他人是否可以查詢你的資料。

**指令：** `/隐私设置 <動作> <目標>`

**動作：**

| 動作 | 說明 |
|------|------|
| `查看` / `show` | 查看目前權限設定 |
| `开启` / `allow` | 允許指定目標查詢 |
| `关闭` / `deny` | 拒絕指定目標查詢 |

**目標：** `score`（成績資料）、`all`（全部）

**示例：**

| 指令 |
|------|
| `/隐私设置 查看` |
| `/隐私设置 关闭 score` |
| `/隐私设置 开启 all` |

## 好友管理

管理你的 McMai 好友清單，好友清單用於 PK 功能。

**指令：** `/好友 [動作] [參數]`

**別名：** `friend`、`好友管理`

**動作：**

| 動作 | 說明 | 參數 |
|------|------|------|
| `列表` | 查看好友清單 | — |
| `添加` | 新增好友 | `#使用者名稱` |
| `删除` | 刪除好友 | `#使用者名稱` 或槽位號 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/好友 列表` | 查看好友清單 |
| `/好友 添加 #Sakura` | 新增名為 Sakura 的好友 |
| `/好友 添加 #Sakura 1` | 在槽位號為 1 的位置覆蓋新增名為 Sakura 的好友 |
| `/好友 删除 1` | 刪除槽位號為 1 的好友 |
