# 定數與表格

## 定數表

查詢指定等級或定數的曲目定數表（圖片形式）。

**指令：** `/定数表 <等級或定數> [伺服器]`

**參數：**

| 參數 | 說明 | 示例 |
|------|------|------|
| 等級或定數 | 等級（如 `13`、`14+`）或定數範圍 | `13+`、`14.0` |
| 伺服器 | 查詢區服（可選） | `cn`、`jp`、`intl` |

**示例：**

| 指令 |
|------|
| `/定数表 13+` |
| `/定数表 14` |
| `/定数表 14 jp` |

---

## 段位表

查詢指定段位的通關曲目要求。

**指令：** `/段位表 <段位名稱> [伺服器] [#使用者名稱]`

**參數：**

| 參數 | 說明 |
|------|------|
| 段位名稱 | 如 `真初段`、`真十段`、`真皆伝` 等 |
| 伺服器 | `cn`、`jp`、`intl`（可選） |
| #使用者名稱 | 可選，查詢他人進度（需對方開啟權限） |

**示例：**

| 指令 |
|------|
| `/段位表 真十段` |
| `/段位表 真十段 #Sakura` |
| `/段位表 真皆伝 jp` |

---

## 完成表

查詢牌子／等級評價的完成進度表。

**指令：** `/完成表 <等級評價或牌子> [牌子等級] [伺服器] [#使用者名稱]`

**示例：**

| 指令 |
|------|
| `/完成表 13` |
| `/完成表 14 #Sakura` |
| `/完成表 真 极` |
| `/完成表 舞将 jp #Sakura` |

---

## 上分查詢

查詢在目前 Rating 下，打哪些曲目可以提升 Rating。

**指令：** `/上分 <等級或定數> <分數> [頁數]`

**參數：**

| 參數 | 說明 | 示例 |
|------|------|------|
| 等級或定數 | 目標等級或定數 | `14`、`13.7` |
| 分數 | 目標分值 | `10` |
| 頁數 | 可選，分頁查詢 | `2` |

**示例：**

| 指令 |
|------|
| `/上分 14 5` |
| `/上分 13.7 9 2` |
