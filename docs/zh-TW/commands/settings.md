# 個人設定

## 頂板查看

查看你的頂板展示（個人最佳成績的卡片式圖片）。

**指令：** `/查看顶板 [區服]`

**參數：**

| 參數 | 說明 |
|------|------|
| 區服 | 可選，指定查詢區服（`cn`、`jp`、`intl`） |

---

## 頂板樣式設定

自訂頂板展示的欄位與風格。

**指令：** `/顶板样式 <動作> <欄位>`

**示例：**

| 指令 | 說明 |
|------|------|
| `/顶板样式 查看` | 查看目前設定 |
| `/顶板样式 开启 游玩次数` | 顯示遊玩次數 |
| `/顶板样式 关闭 昵称` | 隱藏暱稱 |

---

## 隱私設定

控制他人查詢你的資料權限。

**指令：** `/隐私设置 <動作> <目標>`

**動作與目標：**

| 動作 | 說明 |
|------|------|
| `查看` / `show` | 查看目前權限配置 |
| `开启` / `allow` | 允許查詢 |
| `关闭` / `deny` | 禁止查詢 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/隐私设置 查看` | 查看目前權限配置 |
| `/隐私设置 关闭 score` | 禁止他人查詢你的成績 |
| `/隐私设置 开启 all` | 允許查詢所有資料 |
