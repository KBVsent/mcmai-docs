# 個人設定

## 頂板查看

查看你的頂板展示（個人最佳成績的卡片式圖片）。

**指令：** `/查看頂板 [區服]`

**參數：**

| 參數 | 說明 |
|------|------|
| 區服 | 可選，指定查詢區服（`cn`、`jp`、`intl`） |

---

## 頂板樣式設定

自訂頂板展示的欄位與橫幅風格。

**指令：** `/頂板樣式 <動作> [欄位]`

**動作：**

| 動作 | 說明 |
|------|------|
| `查看`（或不填） | 查看目前設定 |
| `顯示` / `show` | 顯示指定欄位 |
| `隱藏` / `hide` | 隱藏指定欄位 |
| `重置` / `reset` | 恢復所有設定為預設值 |

**可用欄位：**

| 欄位名 | 說明 |
|--------|------|
| `暱稱` | 遊戲暱稱 |
| `遊玩次數` | 總遊玩次數 |
| `本版本遊玩次數` | 目前版本遊玩次數 |
| `覺醒數` | 總覺醒數 |
| `里程` | 舞萌里程 |
| `禮物數` | 拍檔禮物數 |
| `小組` | 所在小組 |
| `橫幅` | 頂部橫幅圖 |
| `全部` | 同時操作所有欄位 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/頂板樣式 查看` | 查看目前設定 |
| `/頂板樣式 顯示 遊玩次數` | 顯示遊玩次數 |
| `/頂板樣式 隱藏 暱稱` | 隱藏暱稱 |
| `/頂板樣式 顯示 橫幅` | 顯示頂部橫幅 |
| `/頂板樣式 隱藏 全部` | 隱藏所有欄位 |
| `/頂板樣式 重置` | 恢復預設設定 |

## 隱私設定

控制他人能否查詢你的遊戲資料。詳見[帳號與伺服器 → 隱私設定](/commands/account#隱私設定)。

**指令：** `/隱私設定 <動作> [目標]`

**目標：**

| 目標 | 說明 |
|------|------|
| `查詢` | 總開關，控制所有查詢權限 |
| `b50查詢` | Best 50 等各類 50 查詢 |
| `分數查詢` | 成績（info）查詢 |
| `進度查詢` | 完成表 / 進度表查詢 |

**示例：**

| 指令 | 說明 |
|------|------|
| `/隱私設定 查看` | 查看目前權限配置 |
| `/隱私設定 關閉 查詢` | 禁止他人查詢你的所有資料 |
| `/隱私設定 關閉 分數查詢` | 單獨禁止他人查詢成績 |
| `/隱私設定 重置 查詢` | 恢復所有權限為預設值 |
