# 成績查詢

## 曲目個人遊玩資料

查詢指定曲目的個人遊玩詳情，包括達成率、DX 分、FC/FS 狀態等。

**指令：** `/info <曲目ID或曲名>`

**參數：**

| 參數 | 說明 |
|------|------|
| 曲目ID或曲名 | 數字 ID 或歌曲名稱/別名 |

支援查詢他人資料，在指令後加 `#使用者名稱`。

**示例：**

| 指令 |
|------|
| `/info 11451` |
| `/info 11451 #Sakura` |

---

## 曲目全球統計

查詢指定曲目的全平台（所有玩家）統計資料，包括平均達成率、分布等。

**指令：** `/ginfo <曲目ID或曲名> [難度顏色]`

**參數：**

| 參數 | 說明 | 預設值 |
|------|------|--------|
| 曲目ID或曲名 | 數字 ID 或歌曲名稱 | — |
| 難度顏色 | `绿`、`黄`、`红`、`紫`、`白` | 紫（Master） |

**示例：**

| 指令 |
|------|
| `/ginfo 11451` |
| `/ginfo 11451 白` |

---

## 分數線容錯

查詢指定難度和分數線下，每個 Note 的容錯值。

**指令：** `/分数线 <歌曲ID> [難度顏色]`

**參數：**

| 參數 | 說明 | 預設值 |
|------|------|--------|
| 歌曲ID | 數字 ID | — |
| 難度顏色 | `绿`、`黄`、`红`、`紫`、`白` | 紫 |

**示例：**

| 指令 |
|------|
| `/分数线 11451` |
| `/分数线 11820 白` |

---

## 譜面統計

查詢指定曲目各難度的 AP 率、FC 率、遊玩率等統計資料。

**指令：** `/谱面统计 <曲目ID或曲名> [區服]`

**參數：**

| 參數 | 說明 | 可選值 |
|------|------|--------|
| 曲目ID或曲名 | 數字 ID 或歌曲名稱/別名 | — |
| 區服 | 篩選區服（可選） | `cn`、`jp`、`intl` |

**示例：**

| 指令 |
|------|
| `/谱面统计 11451` |
| `/谱面统计 11451 jp` |
| `/谱面统计 11451 cn` |
