# 成績查詢

## 曲目個人遊玩資料

查詢指定曲目的個人遊玩詳情，包括達成率、DX 分、FC/FS 狀態等。

**指令：** `/info <曲目ID或曲名>`

**參數：**

| 參數 | 說明 |
|------|------|
| 曲目ID或曲名 | 數字 ID 或歌曲名稱/別名 |

支援查詢他人資料，在指令後加 `#使用者名稱`。

**示例：**

| 指令 |
|------|
| `/info 11451` |
| `/info 潘多拉` |
| `/info 11451 #Sakura` |

---

## 分數線容錯

查詢指定難度下每種 Note 的扣分值，以及 DX 星級所需 DX 分數的容錯範圍。

**指令：** `/分數線 <歌曲ID> [難度顏色]`

**參數：**

| 參數 | 說明 | 預設值 |
|------|------|--------|
| 歌曲ID | 數字 ID | — |
| 難度顏色 | `綠`、`黃`、`紅`、`紫`、`白` | 紫 |

**示例：**

| 指令 |
|------|
| `/分數線 11451` |
| `/分數線 11820 白` |

---

## 譜面統計

查詢指定曲目各難度的 AP 率、FC 率、遊玩率等統計資料。

**指令：** `/譜面統計 <曲目ID或曲名> [區服]`

**參數：**

| 參數 | 說明 | 可選值 |
|------|------|--------|
| 曲目ID或曲名 | 數字 ID 或歌曲名稱/別名 | — |
| 區服 | 篩選區服（可選） | `cn`、`jp`、`intl` |

**示例：**

| 指令 |
|------|
| `/譜面統計 11451` |
| `/譜面統計 潘多拉` |
| `/譜面統計 11451 jp` |
| `/譜面統計 11451 cn` |
