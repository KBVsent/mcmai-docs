# 快速開始

McMai 是一個支援國服、日服、國際服三區資料查詢的全功能 maimai DX 輔助 Bot 插件。

## 新增 Bot

McMai 已透過 QQ 機器人平台發布，掃描下方 QR Code 加好友，或將其拉入群組使用：

<img src="/QQBot.png" alt="McMai QQ Bot QR Code" width="200" />

後續將逐步拓展至 Telegram、Discord 等平台。

## 綁定帳號 & 更新資料

使用 Bot 進行個人資料查詢前，需先完成帳號綁定並更新資料。

**日服 / 國際服用戶**

1. 使用 `/绑定` 指令綁定 SEGA ID
2. 綁定成功後，使用 `/更新记录` 指令更新資料

**國服用戶**

直接發送 MAID（`SGWCMAID` 開頭的 84 位字串）即可更新資料。

::: details 如何取得 MAID？
1. 進入「舞萌 | 中二」公眾號
2. 點擊「獲取二維碼」
3. 開啟 QR Code 頁面
4. 長按掃描，複製出現的 `SGWCMAID` 開頭字串
:::

完成綁定並更新資料後，根據實際遊玩區服切換伺服器：

| 指令 | 說明 |
|------|------|
| `/切服 jp` | 切換至日服 |
| `/切服 intl` | 切換至國際服 |
| `/切服 cn` | 切換至國服（預設）|

::: tip 國服用戶
國服用戶使用 `/用户名 [你的QQ]` 綁定 QQ 後，可使用水魚查分器的資料，但部分功能可能受限。
:::

## 基本使用

完成綁定後，可以嘗試以下常用指令：

| 指令 | 說明 |
|------|------|
| `/帮助` | 查看所有可用指令 |
| `/b50` | 生成最高排名的 50 首歌曲圖表 |
| `/今日舞萌` | 查看今日運勢與推薦遊玩曲目 |
