# はじめに

McMai は CN・JP・INTL 三サーバーのデータ照会をサポートする、全機能搭載の maimai DX 補助 Bot プラグインです。

## Bot を追加する

McMai は QQ ボットプラットフォームで公開されています。下記の QR コードをスキャンして Bot を友達に追加するか、グループに招待してご利用ください：

<img src="/QQBot.png" alt="McMai QQ Bot QR コード" width="200" />

今後、Telegram・Discord などのプラットフォームへの対応も予定しています。

## アカウント連携 & データ更新

個人データ照会機能を使うには、事前にアカウント連携とデータ更新が必要です。

**日本サーバー / 国際サーバー ユーザー**

1. `/绑定` を使って SEGA ID を連携する
2. 連携完了後、`/更新记录` でデータを更新する

**CN サーバー ユーザー**

MAID（`SGWCMAID` で始まる 84 文字の文字列）を直接送信するだけでデータが更新されます。

::: details MAID の取得方法
1. WeChat で「舞萌 | 中二」公式アカウントを開く
2. 「获取二维码」（QR コード取得）をタップ
3. QR コードページが表示される
4. QR コードを長押しでスキャンし、表示される `SGWCMAID` 文字列をコピーする
:::

連携・更新後、対象サーバーに切り替えます：

| コマンド | 説明 |
|---------|------|
| `/切服 jp` | 日本サーバーに切り替え |
| `/切服 intl` | 国際サーバーに切り替え |
| `/切服 cn` | 中国サーバーに切り替え（デフォルト）|

::: tip CN サーバー ユーザー
CN ユーザーは `/用户名 [あなたの QQ]` で QQ を連携すると、Diving Fish スコアトラッカーのデータが利用可能になります。ただし一部機能が制限される場合があります。
:::

## 基本的な使い方

設定が完了したら、以下のコマンドをお試しください：

| コマンド | 説明 |
|---------|------|
| `/帮助` | 全コマンド一覧を表示 |
| `/b50` | Top 50 チャートを生成 |
| `/今日舞萌` | 今日の運勢とおすすめ楽曲を表示 |
