# 進捗照会

## レベル進捗

指定したレベルにおける全楽曲の達成率進捗を、達成率/同期率条件や完了状態で絞り込んで照会します。

**コマンド：** `/进度 <レベルまたは定数> <達成率/同期率> <完了/未完了> [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| レベルまたは定数 | `12`、`13+`、`14.0` など | `13+` |
| 達成率/同期率 | 目標達成率または同期率 | `99.5`、`fc`、`ap`、`SSS` |
| 完了/未完了 | `已完成` または `未完成` | `已完成` |
| ページ | 任意、デフォルト 1 ページ目 | `2` |

**グレード基準：**

| グレード | 達成率 |
|---------|--------|
| `S` | ≥ 97.0000% |
| `S+` | ≥ 98.0000% |
| `SS` | ≥ 99.0000% |
| `SS+` | ≥ 99.5000% |
| `SSS` | ≥ 100.0000% |
| `SSS+` | = 101.0000% |

**例：**

| コマンド |
|---------|
| `/进度 13+ SSS` |
| `/进度 14 fc` |
| `/进度 13+ SS 2` |
| `/进度 14 ap 已完成 2` |

## ジャンル進捗

楽曲ジャンル（ポップス・東方 Project など）別に進捗を照会します。

**コマンド：** `/分类进度 <ジャンル> <達成率/同期率> <完了/未完了> [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| ジャンル | 楽曲カテゴリ名 | `动漫`、`东方` |
| 達成率/同期率 | 目標達成率または同期率 | `99.5`、`fc`、`ap`、`SSS` |
| 完了/未完了 | `已完成` または `未完成` | `已完成` |
| ページ | 任意、デフォルト 1 ページ目 | `2` |

**例：**

| コマンド |
|---------|
| `/分类进度 动漫 SSS` |
| `/分类进度 东方 SS 2` |
| `/分类进度 舞萌 fc 已完成 2` |

---

## スコアリスト

指定したレベル/定数における全楽曲のスコアリストを照会します。

**コマンド：** `/分数列表 <レベルまたは定数> [フィルター] [ページ]`

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| レベルまたは定数 | 必須 | `13`、`14+`、`14.0` |
| フィルター | 任意、FC/AP で絞り込み | `fc`、`fc+`（または `fcp`）、`ap`、`寸` |
| ページ | 任意、デフォルト 1 ページ目 | `2` |

**例：**

| コマンド |
|---------|
| `/分数列表 13+` |
| `/分数列表 14 fc` |
| `/分数列表 14 ap 2` |
