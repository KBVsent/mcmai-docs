# 進捗照会

## レベル進捗 / プレート進捗

指定したレベルまたはバージョンプレートの達成率進捗を、完了状態で絞り込んで照会します（ページング対応）。

**コマンド：** `/進捗 <レベル/定数/プレート> <目標> [完了/未完了/未開始] [ページ]`

### レベルモード

| パラメーター | 説明 | 例 |
|------------|------|----|
| レベルまたは定数 | `12`、`13+`、`14.0` など | `13+` |
| 達成率/同期率目標 | 目標達成率、FC/AP、またはグレード名 | `99.5`、`fc`、`ap`、`SSS` |
| 完了/未完了/未開始 | 任意、完了状態で絞り込み | `完了`、`未完了`、`未開始` |
| ページ | 任意、デフォルト 1 ページ目 | `2` |

**グレード基準：**

| グレード | 達成率 |
|---------|--------|
| `S` | ≥ 97.0000% |
| `S+` | ≥ 98.0000% |
| `SS` | ≥ 99.0000% |
| `SS+` | ≥ 99.5000% |
| `SSS` | ≥ 100.0000% |
| `SSS+` | = 101.0000% |

**同期/FC キーワード：** `fc`、`fc+`、`ap`、`ap+`、`fs`、`fs+`、`fdx`、`fdx+`

**例：**

| コマンド |
|---------|
| `/進捗 13+ SSS` |
| `/進捗 14 fc` |
| `/進捗 13+ SS 完了` |
| `/進捗 14 ap 未完了 2` |

### プレートモード

バージョンプレートのコンプリート進捗を照会します（例：`circle sss`、`prism ap`）。

**形式：** `/進捗 <プレート> <プレートレベル>`

| プレートレベル | 要件 |
|-------------|------|
| `fc` | 全楽曲 FC |
| `sss` | 全楽曲 SSS |
| `ap` | 全楽曲 AP |
| `fdx` | 全楽曲 FDX |

プレート文字（`丸`、`鏡` など）または英語バージョン名（`circle`、`prism` など）が使用できます。

**例：**

| コマンド |
|---------|
| `/進捗 circle sss` |
| `/進捗 plus fc` |
| `/進捗 prism ap 2` |

::: tip
英語版バージョン名エイリアスとプレートレベルを指定できます。例：`/進捗 circle sss`。
:::

## ジャンル進捗

楽曲ジャンル（アニメ・東方 Project など）別に進捗を照会します。

**コマンド：** `/ジャンル進捗 <ジャンル> <達成率/同期率目標> [完了/未完了/未開始] [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| ジャンル | 楽曲カテゴリ名 | `アニメ`、`東方`、`オリジナル`、`ボカロ`、`ゲーム`、`ongeki`、`宴` |
| 達成率/同期率目標 | 目標達成率またはグレード | `99.5`、`fc`、`ap`、`SSS` |
| 完了/未完了/未開始 | 任意、完了状態で絞り込み | `完了`、`未完了`、`未開始` |
| ページ | 任意、デフォルト 1 ページ目 | `2` |

**例：**

| コマンド |
|---------|
| `/ジャンル進捗 アニメ SSS` |
| `/ジャンル進捗 東方 SS 2` |
| `/ジャンル進捗 オリジナル fc 完了 2` |

## スコアリスト

指定したレベル/定数における全楽曲のスコアリストを、多次元フィルターとソートで照会します。

**コマンド：** `/スコアリスト [レベルまたは定数] [フィルター...] [ソート] [ページ]`

**フィルター（複数組み合わせ可）：**

| 種別 | 値 |
|------|-----|
| Combo | `ap`、`fc`、`fc+` / `fcp`、`寸`（惜しくもコンボ切れ） |
| Sync | `fs`、`fs+` / `fsp`、`fdx`、`fdx+` / `fdxp` |
| グレード | `sss+`、`sss`、`ss+`、`ss`、`s+`、`s`、`a`、`aa`、`aaa`、`b`、`bb`、`bbb` など |
| 譜面タイプ | `dx`、`sd` |
| 新旧バージョン | `new`（新バージョン）、`old`（旧バージョン） |
| DX 星レベル | `1star` ～ `6star` |

フィルターの前に `=` を付けると完全一致（Combo/Sync のみ）になります。例：`=fc` は FC だけ（FC+ 未満）の譜面のみ表示します。

**ソート：** `X順`（昇順）または `X順+` / `X順-`（方向を指定）

ソート可能フィールド：`ra`（Rating 貢献）、`達成度` / `acc`、`dx`（DX スコア）、`定数` / `ds`

**例：**

| コマンド | 説明 |
|---------|------|
| `/スコアリスト 13+` | 全 13+ 楽曲のスコア一覧 |
| `/スコアリスト 14 fc` | FC 以上で絞り込み |
| `/スコアリスト 14 =fc` | FC のみ（FC+ 未満）を表示 |
| `/スコアリスト 14 ap 2` | AP 以上で絞り込み、2 ページ目 |
| `/スコアリスト 14 dx sss` | DX 譜面 かつ SSS 以上 |
| `/スコアリスト 14 ra順-` | Rating 貢献の降順でソート |
| `/スコアリスト 14 1star` | DX 星レベル 1 で絞り込み |
