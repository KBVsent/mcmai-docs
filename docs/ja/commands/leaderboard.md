# ランキング

## 全体楽曲ランキング

全プレイヤーを対象とした楽曲の達成率ランキングを照会します。

**コマンド：** `/ランキング <曲 ID または曲名> [難易度] [サーバー]`

**パラメーター：**

| パラメーター | 説明 | デフォルト |
|------------|------|----------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 | — |
| 難易度 | `緑`、`黄`、`赤`、`むらさき`、`白` | むらさき（Master） |
| サーバー | `cn`、`jp`、`intl` | 全サーバー |

**例：**

| コマンド |
|---------|
| `/ランキング 11451` |
| `/ランキング 11451 むらさき` |
| `/ランキング 11451 白 jp` |
| `/ランキング 11451 むらさき cn` |

---

## グループ内楽曲ランキング

現在のグループ内プレイヤーの達成率ランキングを照会します。

**コマンド：** `/グルラン <曲 ID または曲名> [難易度] [サーバー]`

パラメーターは `/ランキング` と同じです。

**例：**

| コマンド |
|---------|
| `/グルラン 11451` |
| `/グルラン 11451 白 cn` |

---

## グループ内 Rating ランキング

現在のグループ内プレイヤーの Rating ランキングを照会します。

**コマンド：** `/グループレート [サーバー]`

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| サーバー | サーバーで絞り込み（任意） | `cn`、`jp`、`intl` |

**例：**

| コマンド |
|---------|
| `/グループレート` |
| `/グループレート jp` |

---

## グループランキングに参加

自分を現在のグループのランキングに追加します。参加後、データが `/グルラン` と `/グループレート` に反映されます。

**コマンド：** `/グルラン参加`

---

## グループランキングから退出

現在のグループのランキングから退出します。

**コマンド：** `/グルラン退出`
