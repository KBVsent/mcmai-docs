# ランキング

## 全体楽曲ランキング

全プレイヤーを対象とした楽曲の達成率ランキングを照会します。

**コマンド：** `/排行 <曲 ID または曲名> [難易度] [サーバー]`

**パラメーター：**

| パラメーター | 説明 | デフォルト |
|------------|------|----------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 | — |
| 難易度 | `绿`、`黄`、`红`、`紫`、`白` | 紫（Master） |
| サーバー | `cn`、`jp`、`intl` | 全サーバー |

**例：**

| コマンド |
|---------|
| `/排行 11451` |
| `/排行 11451 紫` |
| `/排行 11451 白 jp` |
| `/排行 11451 紫 cn` |

---

## グループ内楽曲ランキング

現在のグループ内プレイヤーの達成率ランキングを照会します。

**コマンド：** `/群排行 <曲 ID または曲名> [難易度] [サーバー]`

パラメーターは `/排行` と同じです。

**例：**

| コマンド |
|---------|
| `/群排行 11451` |
| `/群排行 11451 白 cn` |

---

## グループ内 Rating ランキング

現在のグループ内プレイヤーの Rating ランキングを照会します。

**コマンド：** `/群rating [サーバー]`

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| サーバー | サーバーで絞り込み（任意） | `cn`、`jp`、`intl` |

**例：**

| コマンド |
|---------|
| `/群rating` |
| `/群rating jp` |

---

## グループランキングに参加

自分を現在のグループのランキングに追加します。参加後、データが `/群排行` と `/群rating` に反映されます。

**コマンド：** `/加入群榜`

---

## グループランキングから退出

現在のグループのランキングから退出します。

**コマンド：** `/退出群榜`
