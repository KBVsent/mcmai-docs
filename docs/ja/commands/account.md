# アカウントとサーバー

## SEGA ID 連携

QQ アカウントと SEGA ID を連携して maimai NET データを照会します。

JP・INTL サーバーのユーザーのみ連携が必要です。CN ユーザーは MAID で直接更新します。

**コマンド：** `/绑定`

**説明：** 専用の連携リンク（または QR コード）が生成されます。ブラウザで開いて認証を完了してください。

::: warning 注意
`/绑定` を実行するたびに新しい認証コードが生成されます。速やかに連携を完了してください。
:::

## サーバー切り替え

現在の対象サーバーを切り替えます。

**コマンド：** `/切服 <サーバー>`

**パラメーター：**

| パラメーター | 説明 |
|------------|------|
| `cn` | 中国サーバー |
| `jp` | 日本サーバー |
| `intl` | 国際サーバー |

**例：**

| コマンド |
|---------|
| `/切服 jp` |
| `/切服 cn` |
| `/切服 intl` |

## ユーザー名設定

他のユーザーがあなたのデータを照会しやすくするためのユーザー名を設定します。

**コマンド：** `/用户名 [ユーザー名|QQ番号|清除]`

**例：**

| コマンド | 説明 |
|---------|------|
| `/用户名 Sakura` | ユーザー名を Sakura に設定 |
| `/用户名 123456789` | QQ 番号をユーザー名として連携 |
| `/用户名` | 現在のユーザー名を確認 |
| `/用户名 清除` | ユーザー名をクリア |

::: warning 注意
ユーザー名をクエリパラメーターとして使用する際は、必ず `#` を先頭に付けてください。

例：`/b50 #Sakura`（`/b50 Sakura` ではありません）
:::

## プライバシー設定

他のユーザーがあなたのゲームデータを照会できるかどうかを制御します。

**コマンド：** `/隐私设置 <アクション> [対象]`

**アクション：**

| アクション | 説明 |
|-----------|------|
| `查看` | 現在のプライバシー設定を確認 |
| `开启` | 指定対象の照会を許可 |
| `关闭` | 指定対象の照会を拒否 |
| `重置` | 指定対象をデフォルト（全許可）にリセット |

**対象：**

| 対象 | 説明 |
|------|------|
| `查询` | マスタースイッチ — すべての照会権限を制御 |
| `b50查询` | Best 50 などの各種 50 系照会 |
| `分数查询` | スコア（info）照会 |
| `进度查询` | 完成表 / 進捗照会 |

::: tip
`查询`（マスタースイッチ）を無効にすると、他のユーザーはあなたのデータを一切照会できなくなります。個別設定より優先されます。
:::

**例：**

| コマンド | 説明 |
|---------|------|
| `/隐私设置 查看` | 現在のプライバシー状況を確認 |
| `/隐私设置 关闭 查询` | 他ユーザーによるすべての照会を拒否 |
| `/隐私设置 关闭 分数查询` | スコア照会のみを拒否 |
| `/隐私设置 开启 b50查询` | Best 50 照会を再び許可 |
| `/隐私设置 重置 查询` | すべての照会権限をデフォルトにリセット |

## フレンド管理

McMai のフレンドリストを管理します（PK 機能で使用）。

**コマンド：** `/好友 [アクション] [パラメーター]`

**エイリアス：** `friend`、`好友管理`

**アクション：**

| アクション | 説明 | パラメーター |
|-----------|------|------------|
| `列表` | フレンドリストを表示 | — |
| `添加` | フレンドを追加 | `#ユーザー名` |
| `删除` | フレンドを削除 | `#ユーザー名` またはスロット番号 |

**例：**

| コマンド | 説明 |
|---------|------|
| `/好友 列表` | フレンドリストを表示 |
| `/好友 添加 #Sakura` | Sakura というフレンドを追加 |
| `/好友 添加 #Sakura 1` | スロット 1 に Sakura を追加 |
| `/好友 删除 1` | スロット 1 のフレンドを削除 |
