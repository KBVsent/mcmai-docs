# アカウントとサーバー

## SEGA ID 連携

QQ アカウントと SEGA ID を連携して maimai NET データを照会します。

JP・INTL サーバーのユーザーのみ連携が必要です。CN ユーザーは MAID で直接更新します。

**コマンド：** `/連携`

**説明：** 専用の連携リンク（または QR コード）が生成されます。ブラウザで開いて認証を完了してください。

::: warning 注意
`/連携` を実行するたびに新しい認証コードが生成されます。速やかに連携を完了してください。
:::

## サーバー切り替え

現在の対象サーバーを切り替えます。

**コマンド：** `/鯖変更 <サーバー>`

**パラメーター：**

| パラメーター | 説明 |
|------------|------|
| `cn` | 中国サーバー |
| `jp` | 日本サーバー |
| `intl` | 国際サーバー |

**例：**

| コマンド |
|---------|
| `/鯖変更 jp` |
| `/鯖変更 cn` |
| `/鯖変更 intl` |

## Aime カード切り替え

日本サーバーアカウントの Aime カードスロットを確認・切り替えます。

::: info 日本サーバー専用
この機能は、日本サーバーアカウントを連携済みの日本サーバー（JP）ユーザーのみ利用できます。
:::

**コマンド：** `/aime [カード番号]`

**エイリアス：** `選卡`、`卡片`、`切卡`

**説明：**

- `/aime`：アカウント内のすべての Aime カードを一覧表示し、現在選択中のカードを Rating・カード種別（標準カード / 無料カード）とともに表示します。
- `/aime <番号>`：指定した番号のカードに切り替えます。
- アカウント内にカードが 1 枚のみの場合は切り替え不要です。

切り替えは次回 `/データ更新` を実行した時に反映されます。

**例：**

| コマンド | 説明 |
|---------|------|
| `/aime` | アカウント内のすべての Aime カードを表示 |
| `/aime 2` | 2 枚目のカードに切り替え |

## ユーザー名設定

`#ユーザー名` で他のユーザーがあなたのデータを照会しやすくするユーザー名を設定します。

**コマンド：** `/名前 [ユーザー名|QQ番号|クリア|clearqq]`

**例：**

| コマンド | 説明 |
|---------|------|
| `/名前 Sakura` | ユーザー名を Sakura に設定 |
| `/名前` | 現在のユーザー名・照会 ID・連携 QQ を確認 |
| `/名前 クリア` | ユーザー名をクリア |
| `/名前 123456789` | QQ 番号を連携（5〜11 桁の数字） |
| `/名前 clearqq` | 連携済みの QQ 番号を解除 |

**照会 ID：**

すべてのユーザーにはシステムが自動付与する**照会 ID**（`#mc` 始まりの短い ID）があります。ユーザー名を設定していなくても、この ID で他のユーザーから照会してもらえます。ユーザー名を設定後は `#ユーザー名` と照会 ID のどちらでも使えます。

::: warning 注意
ユーザー名をクエリパラメーターとして使用する際は、必ず `#` を先頭に付けてください。

例：`/b50 #Sakura`（`/b50 Sakura` ではありません）
:::

::: tip 審査について
ユーザー名は設定後に審査が行われます：

- **審査中**：`#ユーザー名` で照会は可能ですが、表示時にユーザー名が一部マスクされます
- **審査通過**：ユーザー名が完全表示されます
- **審査不合格**：ユーザー名は自動削除されます。再設定が必要です

ユーザー名は **3 日間に 1 回** しか変更できないため、慎重に設定してください。
:::

## プライバシー設定

他のユーザーがあなたのゲームデータを照会できるかどうかを制御します。

**コマンド：** `/公開設定 <アクション> [対象]`

**アクション：**

| アクション | 説明 |
|-----------|------|
| `状態` | 現在のプライバシー設定を確認 |
| `オン` | 指定対象の照会を許可 |
| `off` | 指定対象の照会を拒否 |
| `リセット` | 指定対象をデフォルト（全許可）にリセット |

**対象：**

| 対象 | 説明 |
|------|------|
| `閲覧` | マスタースイッチ — すべての照会権限を制御 |
| `b50閲覧` | Best 50 などの各種 50 系照会 |
| `スコア閲覧` | スコア（info）照会 |
| `進捗閲覧` | コンプリート表 / 進捗照会 |

::: tip
`閲覧`（マスタースイッチ）を無効にすると、他のユーザーはあなたのデータを一切照会できなくなります。個別設定より優先されます。
:::

**例：**

| コマンド | 説明 |
|---------|------|
| `/公開設定 状態` | 現在のプライバシー状況を確認 |
| `/公開設定 off 閲覧` | 他ユーザーによるすべての照会を拒否 |
| `/公開設定 off スコア閲覧` | スコア照会のみを拒否 |
| `/公開設定 オン b50閲覧` | Best 50 照会を再び許可 |
| `/公開設定 リセット` | すべての照会権限をデフォルトにリセット |

## フレンド管理

McMai のフレンドリストを管理します（PK 機能で使用）。

**コマンド：** `/フレンド [アクション] [パラメーター]`

**エイリアス：** `friend`

**アクション：**

| アクション | 説明 | パラメーター |
|-----------|------|------------|
| `リスト` | フレンドリストを表示 | — |
| `追加` | フレンドを追加 | `#ユーザー名` |
| `削除` | フレンドを削除 | `#ユーザー名` またはスロット番号 |

**例：**

| コマンド | 説明 |
|---------|------|
| `/フレンド リスト` | フレンドリストを表示 |
| `/フレンド 追加 #Sakura` | Sakura というフレンドを追加 |
| `/フレンド 追加 #Sakura 1` | スロット 1 に Sakura を追加 |
| `/フレンド 削除 1` | スロット 1 のフレンドを削除 |
