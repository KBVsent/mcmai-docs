# 楽曲検索

## 曲名 / 別名 / ID で検索

曲名・別名・ID で楽曲を検索します。

**コマンド：** `/曲検索 <キーワード>`

**例：**

| コマンド |
|---------|
| `/曲検索 潘多拉` |
| `/曲検索 11451` |

---

## ID 精密検索

ID を使って詳細な譜面情報（難易度・定数・譜面製作者など）を照会します。

**コマンド：** `/id <曲 ID>`

**例：**

| コマンド |
|---------|
| `/id 11451` |

---

## 定数で検索

指定した定数の範囲内にある楽曲一覧を表示します（ページング対応）。

**コマンド：** `/定数検索 <定数> [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| 定数 | 単一値または範囲（`最小-最大`） | `13.7`、`14.0-14.7` |
| ページ | ページ番号（任意、デフォルト 1） | `2` |

**例：**

| コマンド |
|---------|
| `/定数検索 14.0` |
| `/定数検索 13.7-14.0` |
| `/定数検索 14.0 2` |

---

## BPM で検索

指定した BPM 範囲内の楽曲一覧を表示します（ページング対応）。

**コマンド：** `/BPM検索 <BPM> [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| BPM | 単一値または範囲（`最小-最大`） | `180`、`160-200` |
| ページ | ページ番号（任意、デフォルト 1） | `2` |

**例：**

| コマンド |
|---------|
| `/BPM検索 180` |
| `/BPM検索 160-200` |

---

## 作曲者で検索

指定した作曲者・編曲者名の楽曲を検索します。

**コマンド：** `/アーティスト検索 <作曲者名> [ページ]`

**例：**

| コマンド |
|---------|
| `/アーティスト検索 t+pazolite` |
| `/アーティスト検索 xi 2` |

---

## 譜面製作者で検索

指定した譜面製作者名の楽曲を検索します。

**コマンド：** `/デザイナー検索 <譜面製作者名> [ページ]`

**例：**

| コマンド |
|---------|
| `/デザイナー検索 Jack 2` |

---

## バージョン検索

ゲームバージョンまたはリリース日で楽曲を絞り込みます（ページング対応）。

**コマンド：** `/バージョン検索 <バージョンキーワード> [ページ]`

| 入力形式 | 説明 | 例 |
|---------|------|----|
| `new` / `latest` | 自サーバーの最新アップデートで追加された楽曲 | `/バージョン検索 new` |
| 日付 | 指定日にリリースされた楽曲（`YYYYMMDD` または `YYYY-MM-DD`） | `/バージョン検索 2025-01-09` |
| プレート文字 | 対応するゲームバージョンの楽曲 | `/バージョン検索 丸`、`/バージョン検索 鏡` |
| バージョン名部分文字列 | バージョン名のキーワード（大文字・小文字不問） | `/バージョン検索 buddies`、`/バージョン検索 2025` |

**例：**

| コマンド |
|---------|
| `/バージョン検索 new` |
| `/バージョン検索 2025-01-09` |
| `/バージョン検索 丸` |
| `/バージョン検索 prism 2` |

::: tip サーバー横断検索
お使いのサーバーに該当バージョンがない場合は、自動的に他サーバーから該当バージョンの楽曲を検索して返します。結果の上部に実際の取得元サーバーが表示されます。
:::
