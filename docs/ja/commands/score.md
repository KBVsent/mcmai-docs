# スコア照会

## 個人楽曲データ

指定した楽曲の個人プレイ詳細（達成率・DX スコア・FC/FS 状態など）を照会します。

**コマンド：** `/info <曲 ID または曲名>`

| パラメーター | 説明 |
|------------|------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 |

`#ユーザー名` を付けると他のプレイヤーのデータを照会できます。

**例：**

| コマンド |
|---------|
| `/info 11451` |
| `/info 11451 #Sakura` |

---

## 全体楽曲統計

指定した楽曲の全プレイヤー統計データ（平均達成率・分布など）を照会します。

**コマンド：** `/ginfo <曲 ID または曲名> [難易度]`

| パラメーター | 説明 | デフォルト |
|------------|------|----------|
| 曲 ID または曲名 | 数字 ID または曲名 | — |
| 難易度 | `绿`、`黄`、`红`、`紫`、`白` | 紫（Master） |

**例：**

| コマンド |
|---------|
| `/ginfo 11451` |
| `/ginfo 11451 白` |

---

## スコアライン許容ミス数計算

指定した難易度と目標スコアに対して、各ノーツの許容ミス数を計算します。

**コマンド：** `/分数线 <難易度> <曲 ID> <目標%>`

| パラメーター | 説明 |
|------------|------|
| 難易度 | `绿`、`黄`、`红`、`紫`、`白` |
| 曲 ID | 数字 ID |
| 目標% | 目標達成率、例：`101.0` |

**例：**

| コマンド |
|---------|
| `/分数线 紫 11451 101.0` |
| `/分数线 白 11451 100.5` |

---

## 譜面統計

指定した楽曲の各難易度における AP 率・FC 率・プレイ率などの統計を照会します。

**コマンド：** `/谱面统计 <曲 ID または曲名> [サーバー]`

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 | — |
| サーバー | サーバーで絞り込み（任意） | `cn`、`jp`、`intl` |

**例：**

| コマンド |
|---------|
| `/谱面统计 11451` |
| `/谱面统计 11451 jp` |
| `/谱面统计 11451 cn` |
