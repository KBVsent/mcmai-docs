# スコア照会

## 個人楽曲データ

指定した楽曲の個人プレイ詳細（達成率・DX スコア・FC/FS 状態など）を照会します。

**コマンド：** `/info <曲 ID または曲名>`

| パラメーター | 説明 |
|------------|------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 |

`#ユーザー名` を付けると他のプレイヤーのデータを照会できます。

**例：**

| コマンド |
|---------|
| `/info 11451` |
| `/info 潘多拉` |
| `/info 11451 #Sakura` |

---

## スコアライン許容値

指定した難易度における各ノーツ種別の減点値、および各 DX 星レベルに必要な DX スコアの許容範囲を照会します。

**コマンド：** `/ボーダー <曲 ID> [難易度]`

| パラメーター | 説明 | デフォルト |
|------------|------|----------|
| 曲 ID | 数字 ID | — |
| 難易度 | `緑`、`黄`、`赤`、`むらさき`、`白` | むらさき（Master） |

**例：**

| コマンド |
|---------|
| `/ボーダー 11451` |
| `/ボーダー 11820 白` |

---

## 譜面統計

指定した楽曲の各難易度における AP 率・FC 率・プレイ率などの統計を照会します。

**コマンド：** `/譜面統計 <曲 ID または曲名> [サーバー]`

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| 曲 ID または曲名 | 数字 ID または曲名/別名 | — |
| サーバー | サーバーで絞り込み（任意） | `cn`、`jp`、`intl` |

**例：**

| コマンド |
|---------|
| `/譜面統計 11451` |
| `/譜面統計 潘多拉` |
| `/譜面統計 11451 jp` |
| `/譜面統計 11451 cn` |
