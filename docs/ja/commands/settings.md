# 個人設定

## プロフィールカード表示

あなたのプロフィールカード（最高成績のカード形式画像）を表示します。

**コマンド：** `/查看顶板 [サーバー]`

| パラメーター | 説明 |
|------------|------|
| サーバー | 任意、対象サーバーを指定（`cn`、`jp`、`intl`） |

---

## プロフィール表示設定

プロフィールカードに表示するフィールドをカスタマイズします。

**コマンド：** `/顶板样式 <アクション> <フィールド>`

**例：**

| コマンド | 説明 |
|---------|------|
| `/顶板样式 查看` | 現在の設定を表示 |
| `/顶板样式 开启 游玩次数` | プレイ回数を表示 |
| `/顶板样式 关闭 昵称` | ニックネームを非表示 |

---

## プライバシー設定

他のユーザーがあなたのデータを照会できるかどうかを制御します。

**コマンド：** `/隐私设置 <アクション> <対象>`

| アクション | 説明 |
|-----------|------|
| `查看` / `show` | 現在の権限設定を確認 |
| `开启` / `allow` | 照会を許可 |
| `关闭` / `deny` | 照会を拒否 |

**例：**

| コマンド | 説明 |
|---------|------|
| `/隐私设置 查看` | 現在の権限を確認 |
| `/隐私设置 关闭 score` | 他のユーザーによる成績照会を拒否 |
| `/隐私设置 开启 all` | すべてのデータ照会を許可 |
