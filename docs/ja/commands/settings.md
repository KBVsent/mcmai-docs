# 個人設定

## プロフィールカード表示

あなたのプロフィールカード（最高成績のカード形式画像）を表示します。

**コマンド：** `/プロフィール [サーバー]`

| パラメーター | 説明 |
|------------|------|
| サーバー | 任意、対象サーバーを指定（`cn`、`jp`、`intl`） |

---

## プロフィール表示設定

プロフィールカードに表示するフィールドをカスタマイズします。

**コマンド：** `/プロフ設定 <アクション> [フィールド]`

**アクション：**

| アクション | 説明 |
|-----------|------|
| `状態`（または引数なし） | 現在の設定を表示 |
| `表示` / `show` | 指定フィールドを表示 |
| `非表示` / `hide` | 指定フィールドを非表示 |
| `リセット` / `reset` | すべての設定をデフォルトにリセット |

**利用可能なフィールド：**

| フィールド | エイリアス | 説明 |
|-----------|----------|------|
| `ニックネーム` | `nickname` | ゲーム内ニックネーム |
| `プレイ回数` | `playcount` | 総プレイ回数 |
| `本版本游玩次数` | — | 現バージョンのプレイ回数 |
| `覚醒数` | `awake` | 総覚醒数 |
| `マイル` | `mile` | maimai マイレージ |
| `ギフト数` | `gift` | パートナーギフト数 |
| `サークル` | `circle` | サークル / グループ |
| `バナー` | `banner` | トップバナー画像 |
| `全部` | `all` | すべてのフィールドを一括操作 |

**例：**

| コマンド | 説明 |
|---------|------|
| `/プロフ設定 状態` | 現在の設定を表示 |
| `/プロフ設定 表示 プレイ回数` | プレイ回数を表示 |
| `/プロフ設定 非表示 ニックネーム` | ニックネームを非表示 |
| `/プロフ設定 表示 バナー` | トップバナーを表示 |
| `/プロフ設定 非表示 全部` | すべてのフィールドを非表示 |
| `/プロフ設定 リセット` | デフォルトにリセット |

## プライバシー設定

他のユーザーがあなたのデータを照会できるかどうかを制御します。詳細は[アカウントとサーバー → プライバシー設定](/commands/account#プライバシー設定)を参照してください。

**コマンド：** `/公開設定 <アクション> [対象]`

**対象：**

| 対象 | 説明 |
|------|------|
| `閲覧` | マスタースイッチ — すべての照会権限を制御 |
| `b50閲覧` | Best 50 などの各種 50 系照会 |
| `スコア閲覧` | スコア（info）照会 |
| `進捗閲覧` | コンプリート表 / 進捗照会 |

**例：**

| コマンド | 説明 |
|---------|------|
| `/公開設定 状態` | 現在の権限設定を確認 |
| `/公開設定 off 閲覧` | 他ユーザーによるすべての照会を拒否 |
| `/公開設定 off スコア閲覧` | スコア照会のみを拒否 |
| `/公開設定 リセット` | すべての照会権限をデフォルトにリセット |
