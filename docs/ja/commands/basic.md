# 基本機能

## ヘルプ

利用可能なすべてのコマンドを表示します。

**コマンド：** `/帮助`

---

## 今日の運勢

今日の maimai 運勢を表示します（毎日ランダム生成）。

**コマンド：** `/今日舞萌`

---

## ランダム楽曲

条件に合わせてランダムに楽曲を推薦します。

**コマンド：** `/随机歌曲 [タイプ] [色] [レベル]`

**パラメーター：**

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| タイプ | 譜面タイプ（任意） | `dx`、`sd`（または `标准`） |
| 色 | 難易度カラー（任意） | `绿`（緑）、`黄`（黄）、`红`（赤）、`紫`（紫）、`白`（白） |
| レベル | レベル（任意） | `12`、`13+` など |

**例：**

| コマンド |
|---------|
| `/随机歌曲 dx 紫 14` |
| `/随机歌曲 13+` |
| `/随机歌曲 sd` |
