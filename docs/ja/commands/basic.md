# 基本機能

## ヘルプ

利用可能なすべてのコマンドを表示します（画像形式）。

**コマンド：** `/help`

---

## 今日の運勢

今日の maimai 運勢を表示します（毎日ランダム生成）。

**コマンド：** `/今日のmaimai`

---

## ランダム楽曲

条件に合わせてランダムに楽曲を推薦します。

**コマンド：** `/ランダム [タイプ] [色] [レベル]`

**パラメーター：**

| パラメーター | 説明 | 選択肢 |
|------------|------|--------|
| タイプ | 譜面タイプ（任意） | `dx`、`sd` |
| 色 | 難易度カラー（任意） | `緑`、`黄`、`赤`、`むらさき`、`白` |
| レベル | レベル（任意） | `12`、`13+` など |

**例：**

| コマンド |
|---------|
| `/ランダム dx むらさき 14` |
| `/ランダム 13+` |
| `/ランダム sd` |

::: tip
パラメーターの順番は固定ではありません。入力時は任意に組み合わせ可能で、例えば `/ランダム 14 dx` と `/ランダム dx 14` は同じ結果になります。
:::

## 言語切り替え

Bot の返信言語を切り替えます。

**コマンド：** `/lang [言語コード]`

**パラメーター：**

| コード | 言語 |
|--------|------|
| `zh`（または `cn`） | 簡体中文（デフォルト） |
| `zh-Hant`（または `tw`） | 繁體中文（台灣） |
| `zh-Hant-HK`（または `hk`） | 繁體中文（香港） |
| `en`（または `english`） | English |
| `ja`（または `jp`） | 日本語 |

引数なしで実行すると、現在の言語設定を確認できます。

**例：**

| コマンド | 説明 |
|---------|------|
| `/lang` | 現在の言語を確認 |
| `/lang ja` | 日本語に切り替え |
| `/lang en` | 英語に切り替え |
