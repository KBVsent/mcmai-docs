# 譜面定数表

## 定数表

指定したレベルまたは定数の定数表（画像形式）を照会します。

**コマンド：** `/定数表 <レベルまたは定数> [サーバー]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| レベルまたは定数 | レベル（`13`、`14+`）または定数範囲 | `13+`、`14.0` |
| サーバー | 照会サーバー（任意） | `cn`、`jp`、`intl` |

**例：**

| コマンド |
|---------|
| `/定数表 13+` |
| `/定数表 14` |
| `/定数表 14 jp` |

---

## 段位表

指定した段位のコース課題曲情報を照会します。

**コマンド：** `/段位表 <段位名> [サーバー] [#ユーザー名]`

| パラメーター | 説明 |
|------------|------|
| 段位名 | 例：`真初段`、`真十段`、`真皆伝` |
| サーバー | `cn`、`jp`、`intl`（任意） |
| #ユーザー名 | 任意、他のプレイヤーの進捗を照会 |

**例：**

| コマンド |
|---------|
| `/段位表 真十段` |
| `/段位表 真十段 #Sakura` |
| `/段位表 真皆伝 jp` |

---

## コンプリート表

グレード評価またはプレートのコンプリート進捗を照会します。

**コマンド：** `/完成表 <グレードまたはプレート> [プレートレベル] [サーバー] [#ユーザー名]`

**例：**

| コマンド |
|---------|
| `/完成表 SSS` |
| `/完成表 SSS #Sakura` |
| `/完成表 真 极` |
| `/完成表 舞 jp` |

---

## Rating 向上クエリ

現在の Rating を上げるためにプレイすべき楽曲を照会します。

**コマンド：** `/上分 <レベルまたは定数> <スコア> [ページ]`

| パラメーター | 説明 | 例 |
|------------|------|----|
| レベルまたは定数 | 目標レベルまたは定数 | `14`、`13.7` |
| スコア | 目標達成率 | `100.5` |
| ページ | 任意、ページング | `2` |

**例：**

| コマンド |
|---------|
| `/上分 14 5` |
| `/上分 13.7 9 2` |
