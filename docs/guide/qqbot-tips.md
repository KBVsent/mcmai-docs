# QQ Bot 使用小提示

McMai 在 QQ 平台上以 QQ Bot 形式运行，以下是一些实用技巧，帮助你更高效地使用指令。

## 使用 `/` 触发指令菜单

在聊天框中直接输入 `/`，即可唤出 Bot 的指令菜单，快速浏览所有可用指令。

你也可以在 `/` 后继续输入关键词来**筛选指令**，例如输入 `/群排行` 即可快速定位到排行相关的指令。

## `/c` 通用前缀

McMai 新增了 `/c` 作为**通用指令前缀**。

在此之前，群聊中使用 Bot 需要先输入 `@`，从弹出的列表中选择 Bot，再输入指令，流程较为繁琐。

现在，你可以直接在聊天框中输入`/c`，从弹出的指令列表中选择`/c`这条指令之后，继续输入你想要使用的指令和参数

例如：

```
/c 指令名 参数
```

例如，查询歌曲排行：

<img src="/QQbot_Tips_2.jpg" alt="使用 /c 前缀发送指令" style="max-width: 480px;" />

这种方式**无需手动 @ Bot**，大幅简化了群聊中的指令使用体验。

::: tip 小结
- 输入 `/` → 触发指令菜单，支持关键词筛选
- 输入 `/c` → 触发指令菜单，选中 `/c` 指令 → 继续输入`指令`和`参数` → 快速触发不在快捷菜单中的指令
:::
