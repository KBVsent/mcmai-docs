# 快速开始

McMai 是一个支持国服、日服、国际服三区数据查询的全功能 maimai DX 辅助 Bot 插件。

## 添加 Bot

McMai 已通过 QQ 机器人平台发布，扫描下方二维码添加机器人好友，或将其拉入群聊使用。

<img src="/QQBot.png" alt="McMai QQ Bot 二维码" width="200" />

后续将逐步拓展至 Telegram、Discord 等平台。

## 绑定账号 & 更新数据

使用 Bot 进行个人数据查询前，需要先完成账号绑定并更新数据。

**日服 / 国际服用户**

1. 使用 `/绑定` 指令绑定 SEGA ID
2. 绑定成功后，使用 `/更新记录` 指令更新数据

**国服用户**

直接发送 MAID（`SGWCMAID` 开头的 84 位字符串）即可更新数据。

::: details 如何获取 MAID？
1. 进入「舞萌 | 中二」公众号
2. 点击「获取二维码」
3. 打开二维码界面
4. 长按扫描，复制出现的 `SGWCMAID` 开头字符串
:::

完成绑定并更新数据后，根据实际游玩区服切换服务器：

```
/切换服务器 jp     # 切换到日服
/切换服务器 intl   # 切换到国际服
/切换服务器 cn     # 切换到国服（默认）
```

::: tip 国服用户
国服用户使用 `/用户名 [你的QQ]` 绑定 QQ 后，可使用水鱼查分器的数据，但部分功能可能受限。
:::

## 基本使用

完成绑定后，可以尝试以下常用指令：

```
/帮助        # 查看所有可用指令
/b50         # 查询你的 Best 50 成绩
/今日舞萌    # 查看今日运势
```
