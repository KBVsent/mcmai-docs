# Best 50

## Best 50

Query your Best 50 scores (Top 35 SD + Top 15 DX).

**Command:** `/b50`

Add `#username` to query another player's data (requires their permission).

**Examples:**

| Command |
|---------|
| `/b50` |
| `/b50 #SomeUser` |

## PC50

Query your top 50 most-played songs.

This feature is only available for CN region users.
:::

**Command:** `/pc50`

Add `#username` to query another player's data (requires their permission).

---

## PK B50

Compare your B50 against another player's for a side-by-side comparison.

**Command:** `/pk50`

Specify the opponent with `#username`, or select from your friend list.

**Example:** `/pk50 #Sakura`

---

## Update B50 Data

Sync your latest B50 data from maimai NET to local cache.

**Command:** `/更新记录`

::: tip CN Users — Update via QR Code
CN users can use the "Player QR Code" feature from the 舞萌 | 中二 WeChat official account. Long-press the QR code, copy the `SGWCMAID` string, and send it to the Bot to trigger a CN B50 data update.
:::

## Profile Display Settings

Customize the fields shown on your profile card.

**Command:** `/顶板样式 <action> <field>`

**Examples:**

| Command |
|---------|
| `/顶板样式 查看` |
| `/顶板样式 开启 礼物数` |
| `/顶板样式 关闭 版本游玩次数` |
