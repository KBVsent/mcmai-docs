# Best 50

## Best 50

Query your Best 50 scores (Top 35 SD + Top 15 DX).

**Command:** `/b50`

Add `#username` to query another player's data (requires their permission).

**Examples:**

| Command |
|---------|
| `/b50` |
| `/b50 #SomeUser` |

## PK B50

Compare your B50 against another player's, using their B50 songs as the baseline to see score differences on the same charts.

**Command:** `/pk50 [#username]`

- Specify an opponent: `/pk50 #Sakura`
- No argument: randomly match a player from the current group

**Examples:**

| Command | Description |
|---------|-------------|
| `/pk50 #Sakura` | PK against Sakura |
| `/pk50` | Random match in the current group |

---

## Update B50 Data

Sync your latest B50 data from maimai NET to local cache.

**Command:** `/update`

::: tip CN Users — Update via QR Code
CN users can use the "Player QR Code" feature from the 舞萌 | 中二 WeChat official account. Long-press the QR code, copy the `SGWCMAID` string, and send it to the Bot to trigger a CN B50 data update.
:::
