## 随机歌曲

随机推荐一首曲目，可按类型、难度颜色、等级筛选。

**指令：** `/随机歌曲 [类型] [颜色] [等级]`

**参数：**

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 类型 | 曲目类型（可选） | `dx`、`sd`（或`标准`） |
| 颜色 | 难度颜色（可选） | `绿`、`黄`、`红`、`紫`、`白` |
| 等级 | 等级（可选） | `12`、`13+` 等 |

**示例：**

| 指令 |
|------|
| `/随机歌曲 dx 紫 14` |
| `/随机歌曲 13+` |
| `/随机歌曲 sd` |

::: tip
参数顺序不固定，输入时可任意组合，例如 `/随机歌曲 14 dx` 和 `/随机歌曲 dx 14` 效果相同。
:::

## 切换语言

切换 Bot 回复语言。

**指令：** `/lang [语言代码]`

**参数：**

| 语言代码 | 说明 |
|----------|------|
| `zh`（或 `cn`、`简体`） | 简体中文（默认） |
| `zh-Hant`（或 `tw`、`繁体`） | 繁體中文（台灣） |
| `zh-Hant-HK`（或 `hk`、`港繁`） | 繁體中文（香港） |
| `en`（或 `english`） | English |
| `ja`（或 `jp`、`日语`） | 日本語 |

不带参数执行时，显示当前语言设置。

**示例：**

| 指令 | 说明 |
|------|------|
| `/lang` | 查看当前语言 |
| `/lang zh` | 切换到简体中文 |
| `/lang en` | 切换到英文 |
| `/lang ja` | 切换到日文 |
