# 基础功能

## 帮助

查看所有可用指令的列表。

**指令：** `/帮助`

---

## 今日运势

查看今日 maimai 运势，每天随机生成。

**指令：** `/今日舞萌`

---

## 随机歌曲

随机推荐一首曲目，可按类型、难度颜色、等级筛选。

**指令：** `/随机歌曲 [类型] [颜色] [等级]`

**参数：**

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 类型 | 曲目类型（可选） | `dx`、`sd`（或`标准`） |
| 颜色 | 难度颜色（可选） | `绿`、`黄`、`红`、`紫`、`白` |
| 等级 | 等级（可选） | `12`、`13+` 等 |

**示例：**

| 指令 |
|------|
| `/随机歌曲 dx 紫 14` |
| `/随机歌曲 13+` |
| `/随机歌曲 sd` |
