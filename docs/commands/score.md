# 成绩查询

## 曲目个人游玩数据

查询指定曲目的个人游玩详情，包括达成率、DX 分、FC/FS 状态等。

**指令：** `/info <曲目ID或曲名>`

**参数：**

| 参数 | 说明 |
|------|------|
| 曲目ID或曲名 | 数字 ID 或歌曲名称/别名 |

支持查询他人数据，在指令后加 `#用户名`。

**示例：**

| 指令 |
|------|
| `/info 11451` |
| `/info 潘多拉` |
| `/info 11451 #Sakura` |

---

## 曲目全球统计

查询指定曲目的全平台（所有玩家）统计数据，包括平均达成率、分布等。

**指令：** `/ginfo <曲目ID或曲名> [难度颜色]`

**参数：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 曲目ID或曲名 | 数字 ID 或歌曲名称 | — |
| 难度颜色 | `绿`、`黄`、`红`、`紫`、`白` | 紫（Master） |

**示例：**

| 指令 |
|------|
| `/ginfo 11451` |
| `/ginfo 11451 白` |
| `/ginfo 潘多拉 紫` |

---

## 分数线容错

查询指定难度和分数线下，每个 Note 的容错值。

**指令：** `/分数线 <歌曲ID> [难度颜色]`

**参数：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 歌曲ID | 数字 ID | — |
| 难度颜色 | `绿`、`黄`、`红`、`紫`、`白` | 紫 |

**示例：**

| 指令 |
|------|
| `/分数线 11451` |
| `/分数线 11820 白` |

---

## 谱面统计

查询指定曲目各难度的 AP 率、FC 率、游玩率等统计数据。

**指令：** `/谱面统计 <曲目ID或曲名> [区服]`

**参数：**

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 曲目ID或曲名 | 数字 ID 或歌曲名称/别名 | — |
| 区服 | 筛选区服（可选） | `cn`、`jp`、`intl` |

**示例：**

| 指令 |
|------|
| `/谱面统计 11451` |
| `/谱面统计 潘多拉` |
| `/谱面统计 11451 jp` |
| `/谱面统计 11451 cn` |
