# 个人设置

## 顶板查看

查看你的顶板展示（个人最佳成绩的卡片式图片）。

**指令：** `/查看顶板 [区服]`

**参数：**

| 参数 | 说明 |
|------|------|
| 区服 | 可选，指定查询区服（`cn`、`jp`、`intl`） |

---

## 顶板样式设置

自定义顶板展示的字段与风格。

**指令：** `/顶板样式 <动作> <字段>`

**示例：**

| 指令 | 说明 |
|------|------|
| `/顶板样式 查看` | 查看当前设置 |
| `/顶板样式 开启 游玩次数` | 显示游玩次数 |
| `/顶板样式 关闭 昵称` | 隐藏 昵称 |

---

## 隐私设置

控制他人查询你的数据权限。

**指令：** `/隐私设置 <动作> <目标>`

**动作与目标：**

| 动作 | 说明 |
|------|------|
| `查看` / `show` | 查看当前权限配置 |
| `开启` / `allow` | 允许查询 |
| `关闭` / `deny` | 禁止查询 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/隐私设置 查看` | 查看当前权限配置 |
| `/隐私设置 关闭 score` | 禁止他人查询你的成绩 |
| `/隐私设置 开启 all` | 允许查询所有数据 |
