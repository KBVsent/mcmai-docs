# 个人设置

## 顶板查看

查看你的顶板展示（个人最佳成绩的卡片式图片）。

**指令：** `/查看顶板 [区服]`

**参数：**

| 参数 | 说明 |
|------|------|
| 区服 | 可选，指定查询区服（`cn`、`jp`、`intl`） |

---

## 顶板样式设置

自定义顶板展示的字段与横幅风格。

**指令：** `/顶板样式 <动作> [字段]`

**动作：**

| 动作 | 说明 |
|------|------|
| `查看`（或不填） | 查看当前设置 |
| `显示` / `show` | 显示指定字段 |
| `隐藏` / `hide` | 隐藏指定字段 |
| `重置` / `reset` | 恢复所有设置为默认值 |

**可用字段：**

| 字段名 | 说明 |
|--------|------|
| `昵称` | 游戏昵称 |
| `游玩次数` | 总游玩次数 |
| `本版本游玩次数` | 当前版本游玩次数 |
| `觉醒数` | 总觉醒数 |
| `里程` | 舞萌里程 |
| `礼物数` | 拍档礼物数 |
| `小组` | 所在小组 |
| `横幅` | 顶部横幅图 |
| `全部` | 同时操作所有字段 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/顶板样式 查看` | 查看当前设置 |
| `/顶板样式 显示 游玩次数` | 显示游玩次数 |
| `/顶板样式 隐藏 昵称` | 隐藏昵称 |
| `/顶板样式 显示 横幅` | 显示顶部横幅 |
| `/顶板样式 隐藏 全部` | 隐藏所有字段 |
| `/顶板样式 重置` | 恢复默认设置 |

## 隐私设置

控制他人能否查询你的游戏数据。详见[账号与服务器 → 隐私设置](/commands/account#隐私设置)。

**指令：** `/隐私设置 <动作> [目标]`

**目标：**

| 目标 | 说明 |
|------|------|
| `查询` | 总开关，控制所有查询权限 |
| `b50查询` | Best 50 等各类 50 查询 |
| `分数查询` | 成绩（info）查询 |
| `进度查询` | 完成表 / 进度表查询 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/隐私设置 查看` | 查看当前权限配置 |
| `/隐私设置 关闭 查询` | 禁止他人查询你的所有数据 |
| `/隐私设置 关闭 分数查询` | 单独禁止他人查询成绩 |
| `/隐私设置 重置 查询` | 恢复所有权限为默认值 |
