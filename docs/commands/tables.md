# 定数与表格

## 定数表

查询指定等级或定数的曲目定数表（图片形式）。

**指令：** `/定数表 <等级或定数> [服务器]`

**参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 等级或定数 | 等级（如 `13`、`14+`）或定数范围 | `13+`、`14.0` |
| 服务器 | 查询区服（可选） | `cn`、`jp`、`intl` |

**示例：**

| 指令 |
|------|
| `/定数表 13+` |
| `/定数表 14` |
| `/定数表 14 jp` |

---

## 段位表

查询指定段位的通关曲目要求。

**指令：** `/段位表 <段位名称> [服务器] [#用户名]`

**参数：**

| 参数 | 说明 |
|------|------|
| 段位名称 | 如 `真初段`、`真十段`、`真皆伝` 等 |
| 服务器 | `cn`、`jp`、`intl`（可选） |
| #用户名 | 可选，查询他人进度（需对方开启权限） |

**示例：**

| 指令 |
|------|
| `/段位表 真十段` |
| `/段位表 真十段 #Sakura` |
| `/段位表 真皆伝 jp` |

---

## 完成表

查询牌子/等级评价的完成进度表。

**指令：** `/完成表 <等级评价或牌子> [牌子等级] [服务器] [#用户名]`

**示例：**

| 指令 |
|------|
| `/完成表 SSS` |
| `/完成表 SSS #Sakura` |
| `/完成表 真 极` |
| `/完成表 舞 jp` |

---

## 上分查询

查询当前 Rating 下，打哪些曲目可以提升 Rating。

**指令：** `/上分 <等级或定数> <分数> [页数]`

**参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 等级或定数 | 目标等级或定数 | `14`、`13.7` |
| 分数 | 目标达成率 | `100.5` |
| 页数 | 可选，分页查询 | `2` |
**示例：**

| 指令 |
|------|
| `/上分 14 5` |
| `/上分 13.7 9 2` |
