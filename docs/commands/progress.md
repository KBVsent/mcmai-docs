# 进度查询

## 等级进度

查询指定等级下，你的全部曲目达成率进度，支持按评价和分类筛选。

**指令：** `/进度 <等级或定数> <评价> [分类] [页数]`

**参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 等级或定数 | `12`、`13+`、`14.0` 等 | `13+` |
| 评价 | 目标分数线评价 | `SSS`、`SS`、`S`、`A` 等 |
| 分类 | 可选，`ap`、`fc` 等筛选 | `fc` |
| 页数 | 可选，默认第 1 页 | `2` |

**分数评价对照：**

| 评价 | 达成率 |
|------|--------|
| `S` | ≥ 97.0000% |
| `S+` | ≥ 98.0000% |
| `SS` | ≥ 99.0000% |
| `SS+` | ≥ 99.5000% |
| `SSS` | ≥ 100.0000% |
| `SSS+` | = 101.0000% |

**示例：**

| 指令 |
|------|
| `/进度 13+ SSS` |
| `/进度 14 SSS+ fc` |
| `/进度 13+ SS 2` |

## 分类进度

按歌曲分类（流行乐、东方 Project 等）查询进度。

**指令：** `/分类进度 <分类> <评价> [筛选] [页数]`

**参数：**

| 参数 | 说明 |
|------|------|
| 分类 | 歌曲类别名称 |
| 评价 | 目标评价（同等级进度） |
| 筛选 | 可选，`ap`、`fc` 等 |
| 页数 | 可选，默认第 1 页 |

**示例：**

| 指令 |
|------|
| `/分类进度 流行与动漫 SSS` |
| `/分类进度 东方Project SS` |

---

## 分数列表

查询你在指定等级/定数下所有曲目的分数列表。

**指令：** `/分数列表 <等级或定数> [筛选条件] [页数]`

**参数：**

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 等级或定数 | 必填 | `13`、`14+`、`14.0` |
| 筛选条件 | 可选，按 FC/AP 筛选 | `fc`、`fc+`（或 `fcp`）、`ap`、`寸` |
| 页数 | 可选，默认第 1 页 | `2` |

**示例：**

| 指令 |
|------|
| `/分数列表 13+` |
| `/分数列表 14 fc` |
| `/分数列表 14 ap 2` |
