# 进度查询

## 等级进度 / 牌子进度

查询指定等级或版本牌子下的达成进度，支持按完成状态筛选和分页。

**指令：** `/进度 <等级/定数/牌子> <评价/评级> [已完成/未完成/未开始] [页数]`

### 等级进度模式

**参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 等级或定数 | `12`、`13+`、`14.0` 等 | `13+` |
| 评价 | 目标达成率、FC/AP 或评价名 | `99.5`、`fc`、`ap`、`SSS` |
| 已完成/未完成/未开始 | 可选，筛选完成状态 | `已完成`、`未完成`、`未开始` |
| 页数 | 可选，默认第 1 页 | `2` |

**分数评价对照：**

| 评价 | 达成率 |
|------|--------|
| `S` | ≥ 97.0000% |
| `S+` | ≥ 98.0000% |
| `SS` | ≥ 99.0000% |
| `SS+` | ≥ 99.5000% |
| `SSS` | ≥ 100.0000% |
| `SSS+` | = 101.0000% |

**同步/FC 评价：** `fc`、`fc+`、`ap`、`ap+`、`fs`、`fs+`、`fdx`、`fdx+`

**示例：**

| 指令 |
|------|
| `/进度 13+ SSS` |
| `/进度 14 fc` |
| `/进度 13+ SS 已完成` |
| `/进度 14 ap 未完成 2` |

### 版本牌子进度模式

查询指定版本牌子的通关进度（如「丸将」「真极」等）。

**格式：** `/进度 <牌子字> <评级>`

| 评级 | 说明 |
|------|------|
| `极` | FC 所有曲目 |
| `将` | SSS 所有曲目 |
| `神` | AP 所有曲目 |
| `舞舞` | FDX 所有曲目 |

**示例：**

| 指令 |
|------|
| `/进度 丸 将` |
| `/进度 真 极` |
| `/进度 鏡 神 2` |

::: tip
牌子字和评级可以连写，例如 `/进度 丸将` 与 `/进度 丸 将` 效果相同。
:::

## 分类进度

按歌曲分类（动漫、东方 Project 等）查询进度。

**指令：** `/分类进度 <分类> <评价/评级> [已完成/未完成/未开始] [页数]`

**参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 分类 | 歌曲类别名称 | `动漫`、`东方`、`舞萌`、`V家`、`游戏`、`音击`、`宴` |
| 评价/评级 | 目标达成率或评级 | `99.5`、`fc`、`ap`、`SSS` |
| 已完成/未完成/未开始 | 可选，筛选完成状态 | `已完成`、`未完成`、`未开始` |
| 页数 | 可选，默认第 1 页 | `2` |

**示例：**

| 指令 |
|------|
| `/分类进度 动漫 SSS` |
| `/分类进度 东方 SS 2` |
| `/分类进度 舞萌 fc 已完成 2` |

## 分数列表

查询你在指定等级/定数下所有曲目的分数列表，支持多维筛选与排序。

**指令：** `/分数列表 [等级或定数] [筛选条件...] [排序] [页数]`

**筛选条件（可叠加）：**

| 类型 | 可用值 |
|------|--------|
| Combo | `ap`、`fc`、`fc+`/`fcp`、`寸`（寸止） |
| Sync | `fs`、`fs+`/`fsp`、`fdx`、`fdx+`/`fdxp` |
| 评价 | `sss+`、`sss`、`ss+`、`ss`、`s+`、`s`、`a`、`aa`、`aaa`、`b`、`bb`、`bbb` 等 |
| 谱面类型 | `dx`、`sd` |
| 新旧版本 | `new`（新版）、`old`（旧版） |
| DX 星级 | `1star`～`6star` |

在筛选条件前加 `=` 表示精确匹配（仅适用于 Combo/Sync），如 `=fc` 仅显示恰好 FC、未达到 FC+ 的曲目。

**排序：** `按X`（升序）或 `按X+`/`按X-`（指定方向）

可排序字段：`ra`（Rating 贡献）、`达成度`/`acc`、`dx`（DX 分）、`定数`/`ds`

**示例：**

| 指令 | 说明 |
|------|------|
| `/分数列表 13+` | 查询所有 13+ 曲目分数 |
| `/分数列表 14 fc` | 筛选 FC（含 FC+）及以上 |
| `/分数列表 14 =fc` | 仅显示恰好 FC（未达 FC+）的曲目 |
| `/分数列表 14 ap 2` | 筛选 AP 及以上，第 2 页 |
| `/分数列表 14 dx sss` | 同时筛选 DX 谱和 SSS 及以上 |
| `/分数列表 14 按ra-` | 按 Rating 贡献降序排列 |
| `/分数列表 14 1star` | 筛选 DX 星级为 1 星的曲目 |
