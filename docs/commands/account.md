# 账号与服务器

## 绑定 SEGA ID

将你的 QQ 账号与 SEGA ID 关联，用于查询 maimai NET 数据。

只有国际服和日服用户需要绑定账号，国服用户通过 MAID 直接更新数据。

**指令：** `/绑定`

**说明：** 执行后会生成一个专属绑定链接（或二维码），在浏览器中打开并完成授权即可。

::: warning 注意
每次执行 `/bind` 会生成新的验证码，请及时完成绑定。
:::

## 切换服务器

切换你当前绑定的游戏区服。

**指令：** `/切换服武器 <服务器>`

**参数：**

| 参数 | 说明 |
|------|------|
| `cn` | 国服 |
| `jp` | 日服 |
| `intl` | 国际服 |

**示例：**

| 指令 |
|------|
| `/切换服务器 jp` |
| `/切换服务器 cn` |
| `/切换服务器 intl` |

## 设置用户名

设置一个便于他人查询你数据的用户名。

**指令：** `/用户名 [用户名|QQ号|清除]`

**示例：**

| 指令 | 说明 |
|------|------|
| `/用户名 Sakura` | 设置用户名为 Sakura |
| `/用户名 123456789` | 关联 QQ 号为用户名 |
| `/用户名` | 查看当前用户名 |
| `/用户名 清除` | 清除用户名 |

::: warning 注意
在使用使用用户名作为查询参数时，用户名前一定要加`#`。

例如`/b50 #Sakura`，而不是`/b50 Sakura`。
:::

## 隐私设置

控制他人能否查询你的游戏数据。

**指令：** `/隐私设置 <动作> [目标]`

**动作：**

| 动作 | 说明 |
|------|------|
| `查看` | 查看当前隐私设置 |
| `开启` | 允许他人查询指定目标 |
| `关闭` | 禁止他人查询指定目标 |
| `重置` | 恢复指定目标为默认（全部允许） |

**目标：**

| 目标 | 说明 |
|------|------|
| `查询` | 总开关，控制所有查询权限 |
| `b50查询` | Best 50 等各类 50 查询 |
| `分数查询` | 成绩（info）查询 |
| `进度查询` | 完成表 / 进度表查询 |

::: tip
关闭 `查询`（总开关）后，他人将无法查询你的任何数据，优先级高于其他单项设置。
:::

**示例：**

| 指令 | 说明 |
|------|------|
| `/隐私设置 查看` | 查看当前各项隐私状态 |
| `/隐私设置 关闭 查询` | 禁止他人查询你的所有数据 |
| `/隐私设置 关闭 分数查询` | 单独禁止他人查询你的成绩 |
| `/隐私设置 开启 b50查询` | 重新允许他人查询你的 Best 50 |
| `/隐私设置 重置 查询` | 恢复所有查询权限为默认值 |

## 好友管理

管理你的 McMai 好友列表，好友列表用于 PK 功能。

**指令：** `/好友 [动作] [参数]`

**别名：** `friend`、`好友管理`

**动作：**

| 动作 | 说明 | 参数 |
|------|------|------|
| `列表` | 查看好友列表 | — |
| `添加` | 添加好友 | `#用户名` |
| `删除` | 删除好友 | `#用户名` 或槽位号 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/好友 列表` | 查看好友列表 |
| `/好友 添加 #Sakura` | 添加名为 Sakura 的好友 |
| `/好友 添加 #Sakura 1` | 在槽位号为 1 的位置覆盖添加名为 Sakura 的好友 |
| `/好友 删除 1` | 删除槽位号为 1 的好友 |
