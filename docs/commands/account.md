# 账号与服务器

## 绑定 SEGA ID

将你的 QQ 账号与 SEGA ID 关联，用于查询 maimai NET 数据。

只有国际服和日服用户需要绑定账号，国服用户通过 MAID 直接更新数据。

**指令：** `/绑定`

**说明：** 执行后会生成一个专属绑定链接（或二维码），在浏览器中打开并完成授权即可。

::: warning 注意
每次执行 `/bind` 会生成新的验证码，请及时完成绑定。
:::

## 切换服务器

切换你当前绑定的游戏区服。

**指令：** `/切换服武器 <服务器>`

**参数：**

| 参数 | 说明 |
|------|------|
| `cn` | 国服 |
| `jp` | 日服 |
| `intl` | 国际服 |

**示例：**

| 指令 |
|------|
| `/切换服务器 jp` |
| `/切换服务器 cn` |
| `/切换服务器 intl` |

## 设置用户名

设置一个便于他人查询你数据的用户名。

**指令：** `/用户名 [用户名|QQ号|清除]`

**示例：**

| 指令 | 说明 |
|------|------|
| `/用户名 Sakura` | 设置用户名为 Sakura |
| `/用户名 123456789` | 关联 QQ 号为用户名 |
| `/用户名` | 查看当前用户名 |
| `/用户名 清除` | 清除用户名 |

::: warning 注意
在使用使用用户名作为查询参数时，用户名前一定要加`#`。

例如`/b50 #Sakura`，而不是`/b50 Sakura`。
:::

## 隐私设置

控制他人是否可以查询你的数据。

**指令：** `/隐私设置 <动作> <目标>`

**动作：**

| 动作 | 说明 |
|------|------|
| `查看` / `show` | 查看当前权限设置 |
| `开启` / `allow` | 允许指定目标查询 |
| `关闭` / `deny` | 拒绝指定目标查询 |

**目标：** `score`（成绩数据）、`all`（全部）

**示例：**

| 指令 |
|------|
| `/隐私设置 查看` |
| `/隐私设置 关闭 score` |
| `/隐私设置 开启 all` |

## 好友管理

管理你的 McMai 好友列表，好友列表用于 PK 功能。

**指令：** `/好友 [动作] [参数]`

**别名：** `friend`、`好友管理`

**动作：**

| 动作 | 说明 | 参数 |
|------|------|------|
| `列表` | 查看好友列表 | — |
| `添加` | 添加好友 | `#用户名` |
| `删除` | 删除好友 | `#用户名` 或槽位号 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/好友 列表` | 查看好友列表 |
| `/好友 添加 #Sakura` | 添加名为 Sakura 的好友 |
| `/好友 添加 #Sakura 1` | 在槽位号为 1 的位置覆盖添加名为 Sakura 的好友 |
| `/好友 删除 1` | 删除槽位号为 1 的好友 |
