# 账号与服务器

## 绑定 SEGA ID

将你的 QQ 账号与 SEGA ID 关联，用于查询 maimai NET 数据。

只有国际服和日服用户需要绑定账号，国服用户通过 MAID 直接更新数据。

**指令：** `/绑定`

**说明：** 执行后会生成一个专属绑定链接（或二维码），在浏览器中打开并完成授权即可。

::: warning 注意
每次执行 `/bind` 会生成新的验证码，请及时完成绑定。
:::

## 切换服务器

切换你当前绑定的游戏区服。

**指令：** `/切换服务器 <服务器>`

**参数：**

| 参数 | 说明 |
|------|------|
| `cn` | 国服 |
| `jp` | 日服 |
| `intl` | 国际服 |

**示例：**

| 指令 |
|------|
| `/切换服务器 jp` |
| `/切换服务器 cn` |
| `/切换服务器 intl` |

## 切换 Aime 卡片

查看或切换日服账号下的 Aime 卡片槽位。

::: info 仅日服
此功能仅日服（JP）用户可用，且需已绑定日服账号。
:::

**指令：** `/aime [卡片序号]`

**别名：** `选卡`、`卡片`、`切卡`

**说明：**

- `/aime`：列出当前账号下的所有 Aime 卡片，标注当前选中的卡片、Rating 与卡片类型（标准卡 / 免费卡）。
- `/aime <序号>`：切换到指定序号的卡片。
- 若账号下只有一张卡片，则无需切换。

切换后将在下次执行 `/更新记录` 时生效。

**示例：**

| 指令 | 说明 |
|------|------|
| `/aime` | 查看账号下所有 Aime 卡片 |
| `/aime 2` | 切换到第 2 张卡片 |

## 设置用户名

设置一个便于他人通过 `#用户名` 查询你数据的标识符。

**指令：** `/用户名 [用户名|QQ号|清除|clearqq]`

**示例：**

| 指令 | 说明 |
|------|------|
| `/用户名 Sakura` | 设置用户名为 Sakura |
| `/用户名` | 查看当前用户名、查分器 ID 与绑定 QQ |
| `/用户名 清除` | 清除用户名 |
| `/用户名 123456789` | 绑定 QQ 号（5～11 位数字） |
| `/用户名 clearqq` | 解绑已绑定的 QQ 号 |

**查询标识符说明：**

每位用户都有一个系统自动分配的**查分器 ID**（`#mc` 开头的短 ID），即使未设置用户名也可使用它被他人查询。设置用户名后，他人可以通过 `#用户名` 查询你，也可以继续使用查分器 ID。

::: warning 注意
在使用用户名作为查询参数时，用户名前一定要加 `#`。

例如 `/b50 #Sakura`，而不是 `/b50 Sakura`。
:::

::: tip 审核机制
用户名设置后需经过审核：

- **审核中**：可以用 `#用户名` 被他人查询，但显示时用户名会被部分遮盖
- **审核通过**：用户名完整显示
- **审核不通过**：用户名将被自动删除，需重新设置

用户名每 **3 天** 只能修改一次，请谨慎设置。
:::

## 隐私设置

控制他人能否查询你的游戏数据。

**指令：** `/隐私设置 <动作> [目标]`

**动作：**

| 动作 | 说明 |
|------|------|
| `查看` | 查看当前隐私设置 |
| `开启` | 允许他人查询指定目标 |
| `关闭` | 禁止他人查询指定目标 |
| `重置` | 恢复指定目标为默认（全部允许） |

**目标：**

| 目标 | 说明 |
|------|------|
| `查询` | 总开关，控制所有查询权限 |
| `b50查询` | Best 50 等各类 50 查询 |
| `分数查询` | 成绩（info）查询 |
| `进度查询` | 完成表 / 进度表查询 |

::: tip
关闭 `查询`（总开关）后，他人将无法查询你的任何数据，优先级高于其他单项设置。
:::

**示例：**

| 指令 | 说明 |
|------|------|
| `/隐私设置 查看` | 查看当前各项隐私状态 |
| `/隐私设置 关闭 查询` | 禁止他人查询你的所有数据 |
| `/隐私设置 关闭 分数查询` | 单独禁止他人查询你的成绩 |
| `/隐私设置 开启 b50查询` | 重新允许他人查询你的 Best 50 |
| `/隐私设置 重置 查询` | 恢复所有查询权限为默认值 |

## 好友管理

管理你的 McMai 好友列表，好友列表用于 PK 功能。

**指令：** `/好友 [动作] [参数]`

**别名：** `friend`、`好友管理`

**动作：**

| 动作 | 说明 | 参数 |
|------|------|------|
| `列表` | 查看好友列表 | — |
| `添加` | 添加好友 | `#用户名` |
| `删除` | 删除好友 | `#用户名` 或槽位号 |

**示例：**

| 指令 | 说明 |
|------|------|
| `/好友 列表` | 查看好友列表 |
| `/好友 添加 #Sakura` | 添加名为 Sakura 的好友 |
| `/好友 添加 #Sakura 1` | 在槽位号为 1 的位置覆盖添加名为 Sakura 的好友 |
| `/好友 删除 1` | 删除槽位号为 1 的好友 |
