# 排行榜

## 全局单曲排行

查询全平台（所有玩家）的单曲达成率排名。

**指令：** `/排行 <曲目ID或曲名> [难度颜色] [区服]`

**参数：**

| 参数 | 说明 | 默认值 |
|------|------|-----|
| 曲目ID或曲名 | 数字 ID 或歌曲名称/别名 | — |
| 难度颜色 | `绿`、`黄`、`红`、`紫`、`白` | 紫（Master） |
| 区服 | `cn`、`jp`、`intl` | 全区 |

**示例：**

| 指令 |
|------|
| `/排行 11451` |
| `/排行 潘多拉` |
| `/排行 11451 紫` |
| `/排行 11451 白 jp` |
| `/排行 11451 紫 cn` |

---

## 群内单曲排行

查询当前群内已加入排行榜的玩家的单曲达成率排名。

**指令：** `/群排行 <曲目ID或曲名> [难度颜色] [区服]`

**参数与 `/排行` 相同。**

**示例：**

| 指令 |
|------|
| `/群排行 11451` |
| `/群排行 潘多拉 紫` |
| `/群排行 11451 白 cn` |

---

## 群内 Rating 排行

查询当前群内玩家的 Rating 排名。

**指令：** `/群rating [区服]`

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 区服 | 筛选区服（可选） | `cn`、`jp`、`intl`（或中文名） |

**示例：**

| 指令 |
|------|
| `/群rating` |
| `/群rating jp` |
| `/群rating 国服` |

---

## 加入群榜

将自己加入当前群的排行榜，加入后数据才会出现在 `/grank` 和 `/grating` 中。

**指令：** `/加入群榜`

---

## 退出群榜

从当前群的排行榜中退出。

**指令：** `/退出群榜`
